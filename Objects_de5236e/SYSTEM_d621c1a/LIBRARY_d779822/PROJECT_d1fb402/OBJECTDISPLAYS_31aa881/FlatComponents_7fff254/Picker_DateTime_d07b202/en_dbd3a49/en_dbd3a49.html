Picker Date Time

Opens a dialog to pick a date and time.

Parameters

  • nameString=pickerDate
    The control's name. Must be unique in every display if controlled via triggers.
  • Apprearance
    • background colorColor
      Background color. Defaults to ControlBackgroundColor provided by the Theme QD.
    • fill color inactiveColor
      Background color when inactive. Defaults to ControlInactiveBackgroundColor provided by the Theme QD.
    • stroke color pressedColor
      Stroke color when pressed. Defaults to ControlTintColor provided by the Theme QD.
    • focus stroke colorColor
      Stroke color when focussed. Defaults to ControlTintColor provided by the Theme QD.

Events

  • valuechanged: triggered after value change.
