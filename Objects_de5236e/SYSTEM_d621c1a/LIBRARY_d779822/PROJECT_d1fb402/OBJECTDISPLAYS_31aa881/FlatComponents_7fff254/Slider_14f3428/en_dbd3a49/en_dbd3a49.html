Slider

Allows editing a numeric value by dragging up and down.

Parameters

  • baseAddress
    The value to be edited.
  • minimumNumber=0
    The minimum value to pick.
  • maximumNumber=100
    The maximum value to pick.
  • decimal positionsNumber=0
    Precision of the value to pick.
  • tint colorColor
    Indicator color. Defaults to ControlTintColor provided by Theme QD.
  • background colorColor
    Background color. Defaults to ControlIndicatorBackgroundColor provided by Theme QD.
