Description

This control is a simple button which just fires the trigger "clicked". This trigger can be used to add a Simple Dynamic to this button which reacts on the trigger "clicked" and fires a new trigger. This new trigger can then be used by other elements.

Parameters

This control has the following generic parameters:

  • label: label of the button if "visibility label" is set to visible
Font:
  • family: font family which will be applied to label (default: Arial)
  • size: font size which will be applied to the label (default: 16)
  • color: color which will be applied to the label (default: #ffffff)
Appearance:
  • visibility label: if set to visible, the text as specified by "label" will be displayed; if this is set to visible, "visibility symbol'" should be set to hidden. (default: visible)
  • visibility symbol: if set to visible, the symbol as specified by "symbol" will be displayed; if this is set to visible, "visibility label'" should be set to hidden. (default: hidden)
  • symbol: symbol to be used if "visibility symbol" is set to visible; further scales can be selected from "Object Displays (ATVISE)/Symbols/20x20 (default: check)
  • symbol color: color of the symbol to be used  (default: #ffffff)
  • fill color: button's fill color (default: #3f434a)
  • fill color when inactive: button's fill color if not active (default: #8c8c8c)
  • stroke color pressed: the color of the stroke which will be set when the button is pressed (default: #E3D700)
  • focus stroke color: the color of the stroke which will be set when the button is focused (default: #E3D700)
Security:
  • necessary right: the user must have this right to activate this button, i.e. to press the button
  • activation address: if the value of this address equals to "activation value", then the button will be active, i.e. can be pressed
  • activation value: value which activiates the button
NOTE: The activation of a button can be configured only by the necessary right, or only by the activation address and value or by both. In case both settings are made, the user must have the necessary right and the value of the "activation address" must equal the "activation value"; otherwise the button will not be activated.

Consistency:
  • group: name of the consistency group
  • function: function to be executed on click when a consistency group is given (default: write)
Options:
  • tab index: defines the order for changing between the object displays by using the tab key
  • tooltip: displays a tooltip box with the entered text (standard HTML-Elements can be entered)
