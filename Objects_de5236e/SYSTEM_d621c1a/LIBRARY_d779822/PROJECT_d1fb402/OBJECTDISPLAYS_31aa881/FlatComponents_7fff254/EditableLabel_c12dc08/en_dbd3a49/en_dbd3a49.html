Editable Label

Sets/displays a value to/from an address.

Parameters

  • baseAddress
    The value to be displayed/edited.
  • input modeNumber|Boolean|String|Password=Number
    The mode to validate the input against.
  • allow direct inputYes|No=Yes
    Sets if the label should be editable with hardware keyboard or onscreen keyboard.
  • Number Format
    • displayed unit[String]
      The value's unit.
    • show signBoolean=false
      Whether to show the number sign for numbers greater than zero.
    • leading zerosNumber=0
      How many leading zeros to display.
    • post decimal positionsNumber=0
      How many post decimal positions to display
    • force decimal fraction position[Number]
      Missing description
  • Enum List Mode
    • enum list[Global]
      An address pointing to an enum list in the global parameters.
  • Font
    • family<FontFamily>
      The control's font family.
    • size<FontSize>=16
      The control's font size.
    • colorColor
      The control's font color.
    • editing font family<FontFamily>
      The control's font color when editing.
  • Appearance
    • fill colorColor
      The control's background color.
    • fill not editableColor
      The control's background color if not editable.
    • fill color when inactiveColor
      The control's background color if inactive.
    • focus stroke colorColor
      The control's stroke color if focussed.
    • editing background colorColor
      The control's background color when editing.
    • editing border colorColor
      The control's stroke color when editing.
  • Limits
    • limit low lowString=off
      Limit low low (off = no limit).
    • limit lowString=off
      Limit low (off = no limit).
    • limit highString=off
      Limit high (off = no limit).
    • limit high highString=off
      Limit high high (off = no limit).
    • limit low low colorColor=red
      Limit low low color.
    • limit low colorColor=yellow
      Limit low color.
    • limit high colorColor=yellow
      Limit high color.
    • limit high high colorColor=red
      Limit high high color.
    • minimumNumber=off
      Limits minimum (off = no minimum).
    • maximumNumber=off
      Limits maximum (off = no maximum).
    • address for minimum[Address]
      Address pointing to the minimum value.
    • address for maximum[Address]
      Address pointing to the maximum value.
    • address for limit low low[Address]
      Address pointing to the limit low low value.
    • address for limit low[Address]
      Address pointing to the limit low value.
    • address for limit high[Address]
      Address pointing to the limit high value.
    • address for limit high high[Address]
      Address pointing to the limit high high value.
  • Security
    • editableYes|No=Yes
      Sets if the input is editable.
    • necessary right[Address]
      Points to the rights necessary to use this control.
    • activation address[Address]
      Activation address.
    • activation valueString=off
      Activation value.
  • Options
    • password modeYes|No=No
      Sets if in password mode.
    • alternative output address[Address]
      Sets an alternative output address.
    • consistency group[String]
      Consistency group.
    • tab index[Number]
      Tab index.
    • tooltip[TRString]
      Localized Tooltip.
    • select text on focusBoolean=false
      Set if the control selects its text on focus.
