Circular Slider

Allows editing a numeric value by dragging or via keyboard.

Parameters

  • label[TRString]
    The label's text.
  • Value
    • minimumNumber=0
      The minimum value.
    • maximumNumber=100
      The maximum value.
    • show valueBool=true
      Sets if the value shall be displayed.
    • desired value[Address]
      If set, the slider will scale depending this value, the knob will move depending on the base parameter.
  • Number Format
    • displayed unitString
      Unit to be displayed after the value.
    • show signBool=false
      Whether to show the plus/minus sign in front of the value or not.
    • leading zerosNumber=0
      If the value consists of less digits than this value, it gets filled up with zeroes on the left side.
    • post decimal positionsNumber=0
      Number of digits after the decimal mark.
    • force decimal fraction positionNumber=0
      Amount, by which the decimal mark is offset.
  • Font
    • family<FontFamily>
      Font family. Defaults to FontFamily provided by the Theme QD.
    • size<FontSize>=16
      Font size.
  • Colors
    • color gaugeColor
      The gauge's fill color. Defaults to ControlTintColor provided by the Theme QD.
    • color backgroundColor
      Background color. Defaults to ControlBackgroundColor provided by the Theme QD.
    • color labelColor
      The label's font color. Defaults to ControlLabelColor provided by the Theme QD.
    • color gauge inactiveColor
      The gauge's fill color when inactive. Defaults to ControlInactiveBackgroundColor provided by the Theme QD.
  • Limit
    • limit low lowString=off
      Limit low low (off = no limit)
    • limit lowString=off
      Limit low (off = no limit)
    • limit highString=off
      Limit high (off = no limit)
    • limit high highString=off
      Limit high high (off = no limit)
    • limit low low colorColor=red
      Limit low low color
    • limit low colorColor=yellow
      Limit low color
    • limit high colorColor=yellow
      Limit high color
    • limit high high colorColor=red
      Limit high high color
