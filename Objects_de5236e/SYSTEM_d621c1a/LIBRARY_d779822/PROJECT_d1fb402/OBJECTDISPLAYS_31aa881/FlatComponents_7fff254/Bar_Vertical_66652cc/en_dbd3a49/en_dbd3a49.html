Bar Vertical

Displays a value as a bar chart

Parameters

  • Colors
    • tint colorColor
      Tint color. Defaults to ControlTintColor provided by the Theme QD.
    • background colorColor
      Background color. Defaults to ControlBackgroundColor provided by the Theme QD.
    • scale and label colorColor
      Scale and label color. Defaults to ControlLabelColor provided by the Theme QD.
    • bar backgrund colorColor
      Bar background color. Defaults to ControlIndicatorBackgroundColor provided by the Theme QD.
  • Options
    • minimumNumber=0
      The minimum value to display.
    • maximumNumber=100
      The maximum value to display.
    • decimal placesNumber=0
      How many decimal places should be displayed.
  • Label
    • value label colorColor
      Value label font color. Defaults to ControlTextColor provided by the Theme QD.
    • label text[TRString=Label]
      The label's text.
    • label size[Number]
      The size of the label.
  • Font
    • font familyFont Face
      Font face used for the object display.
    • font size[Number]
      The label's text.
  • unit[String]
    The value's unit.
  • desired value[Address]
    If set, a small indicator on the left side will display this value.
