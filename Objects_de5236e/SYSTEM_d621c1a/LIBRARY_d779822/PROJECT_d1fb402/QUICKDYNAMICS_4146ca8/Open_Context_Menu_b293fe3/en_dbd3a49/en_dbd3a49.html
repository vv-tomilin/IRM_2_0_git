Description - Open Context Menu

This Quick Dynamic opens a context menu at the specified element.

Parameters:

  • x coordinate: The x coordinate for the menu (absolute to the display).
  • y coordinate: The y coordinate for the menu (absolute to the display).
  • first, second, third, fourth, fifth entry key: text for the entries 1-5
  • first, second, third, fourth, fifth entry value: value for the entries 1-5
  • trigger name: Name of the trigger that will receive the selected value.
  • output node: The selected value will be written to this node.
  • event triggering the change: the event triggering this Quick Dynamic, which is one of the following:
    • click: mouse click
    • dblclick: double-click of mouse button
    • mousedown: press down the mouse button
    • mouseup: release the mouse button
    • mouseover: move the mouse cursor over the applied graphic element
    • mousemove: move the mouse anywhere over the applied graphic element
    • mouseout: move the mouse cursor out of the applied graphic element
  • maximum displayed items (0 = all):
    • With this parameter you can set the amount of items shown in the dropdown field.
    • 0 = show all
    • Hint: If there are more items in the list than displayed, you can browse through the list pagewise.
  • font size: Sets the font size of the items in the dropdown field.
  • minimum width of the PopUp: Sets the minimum width of the menu.
