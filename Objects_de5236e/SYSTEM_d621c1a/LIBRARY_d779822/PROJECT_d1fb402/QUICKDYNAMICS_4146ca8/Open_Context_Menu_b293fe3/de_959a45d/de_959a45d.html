Beschreibung - Kontextmenü öffnen

Dieses Quick Dynamic öffnet ein Kontextmenü am vorgegebenen Element.

Parameter:

  • X Koordinate: Die x-Koordinate für das Menü (absolut zum Display)
  • Y Koordinate: Die Y-Koordinate für das Menü (absolut zum Display)
  • Erster, Zweiter, Dritter, Vierter, Fünfter Eingabe-Schlüssel: Text für die Eingaben 1-5
  • Erster, Zweiter, Dritter, Vierter, Fünfter Eingabewert: Wert für die Eingaben 1-5
  • Triggername: Name des Triggers, der den ausgewählten Wert empfängt
  • Ausgabe-Datenpunkt: Der ausgewählte Wert wird in diesen Datenpunkt geschrieben.
  • Ereignis, das die Änderung triggert: Das Ereignis, das dieses Quick Dynamic aktiviert, wobei es sich um eines der folgenden handelt:
    • click: Mausklick
    • dblclick: Doppelklicken mit der Maustaste
    • mousedown: Drücken der Maustaste
    • mouseup: Loslassen der Maustaste
    • mouseover: Bewegen des Mauszeigers über das angewendete grafische Element
    • mousemove: Bewegen der Maus an beliebiger Stelle über das angewendete grafische Element
    • mouseout: Bewegen des Mauszeigers aus dem angewendeten grafischen Element heraus
  • Maximal angezeigte Elemente (0 = alle):
    • Mit diesem Parameter kann die Anzahl der gleichzeitig angezeigten Einträge im Dropdownfeld festgelegt werden.
    • 0 = alle anzeigen
    • Hinweis: Gibt es mehr Einträge, als jene die gleichzeitig angezeigt werden, so können Sie seitenweise durchblättern.
  • Schriftgröße: Gibt die Schriftgröße für die Einträge im Dropdownfeld an.
  • Minimale Breite des PopUp's: Mit diesem Paramater wird die minimale Breite des Menüs definiert.
