Beschreibung - TreeView

Dieses Quick Dynamic erstellt eine Baumansicht aufgrund einer opcUA Nodestruktur.

Die folgenden Optionen und Methoden werden bereitgestellt:

Options
Name Beschreibung Defaultwert Erforderlich
renderTo Die id des DIV-Elements in dem die Baumstruktur erstellt werden soll x
busyIndicatorTolerance Die Zeit die vergehen darf, bis der BusyIndicator angezeigt wird [ms] 100
expandoOffset Die Distanz vom Expando-Element zum Icon des Zweigs [px] 3
fontSize Die Schriftgröße für die Baumstruktur [px] 12
searchDepth Die maximale Ordnertiefe in der die Filterfunktion sucht 100
imagePaths Die Pfade zu den Icons die in der Baumstruktur angezeigt werden {
baseVariableImage: "/treeView/icons/baseVariable.svg",
folderImage: "/treeView/icons/folder.svg",
aggregateFunctionImage: "/treeView/icons/aggregateFunction.svg",
collapsedLeafImage: "/treeView/icons/leafCollapsed.svg",
expandedLeafImage: "/treeView/icons/leafExpanded.svg",
}
leafIndentation Die Einrückung für Baumzweige [px] 15
leafPadding Das Padding oben und unten an den Baumzweigen [px] 2
selectableTypes Die Typen von Baumzweigen die selektierbar sein sollen ["baseVariable", "aggregateFunction"]
startAddress Die opcUA Adresse die für die Baumstruktur als Ausgangspunkt verwendet werden soll "AGENT.OBJECTS"
