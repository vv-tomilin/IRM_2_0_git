Description - TreeView

This Quick Dynamic creates a treeView based on a opcUA node structure.

The following options and methods are provided:

Options
Name Description Default Value Required
renderTo The id of the DIV-element the tree should be created in x
busyIndicatorTolerance The time to has to pass by to start displaying the busy indicator [ms] 100
expandoOffset The distance from the expando to the leaf icon[px] 3
fontSize The font size for the treeView [px] 12
searchDepth The maximal folderdepth the filter method is searching at 100
imagePaths The image paths for the tree icons {
baseVariableImage: "/treeView/icons/baseVariable.svg",
folderImage: "/treeView/icons/folder.svg",
aggregateFunctionImage: "/treeView/icons/aggregateFunction.svg",
collapsedLeafImage: "/treeView/icons/leafCollapsed.svg",
expandedLeafImage: "/treeView/icons/leafExpanded.svg",
}
leafIndentation The indentation for child leaves [px] 15
leafPadding The padding top and bottom between the leaves [px] 2
selectableTypes The types of leaves that can be selected ["baseVariable", "aggregateFunction"]
startAddress The opcUA address to start displaying the tree from "AGENT.OBJECTS"
Methods
Name Description Parameters Return Value
collapse Collapses a leaf -node
(HTMLElement of the leaf to collapse)
collapseAll Collapses a leaves
expand Expands a leaf -node
(HTMLElement of the leaf to expand)
expandAll Expands all leaves
getSelectedNode Get the data for the currently selected tree element (node) data object
selectNode Select a tree element (node) -nodePath (z.B.: "AGENT.OBJECT.folder1.myNode")
-aggregatePath (z.B.: "AGENT.HISTORY.AGGREGATETEMPLATES.myTemplate.myAverage")
filter Only show tree elements (nodes), that contain a specific search string. -searchString (z.B.: "myNode")
-refreshSearchData (renew list of available nodes)
off Detach an event handler from the tree view -name
-callbackFunction
on Attach an event handler to the tree view
Available Events
expand, expandAll, collapse, collapseAll, select, render, ready		 -name
-callbackFunction
-callerScope
destroy Destroy the treeView and release memory
