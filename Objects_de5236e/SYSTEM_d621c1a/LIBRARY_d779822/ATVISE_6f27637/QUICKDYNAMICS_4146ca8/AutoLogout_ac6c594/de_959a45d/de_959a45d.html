Beschreibung - Automatisches Ausloggen

Dieses Quick Dynamic ermöglicht die Auto Logout Funktionalität.

Achtung: Die Auto Logout Funktionalität kann über das Menü aus den Quick Dynamics ausgewählt werden. Dies sollte idealerweise auf dem Default Display erfolgen. Parameter können nachträglich geändert werden, indem die Quick Dynamic "Automatisches Ausloggen" aus dem Bereich Dynamisierung ausgewählt wird.

Parameter:

  • Zeitspanne für das Logout in Sekunden: Dieser Parameter definiert die Zeitspanne in Sekunden, nach welcher das automatische Ausloggen erfolgt.
  • Zielframe: Dieser Parameter definiert den Zielframe.
  • Auslogg-Display: Dieser Parameter definiert das Display, welches nach dem automatischen Ausloggen angezeigt wird.
