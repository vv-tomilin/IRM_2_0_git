Description - Auto Logout

This Quick Dynamic enables the auto logout functionality.

Please note: If you want to use the auto logout functionality you must enable this Quick Dynamic by choosing it in the top menu from the Quick Dynamics ideally on your default display. If you want to change some parameter values afterwards then you can do it by editing the Quickdynamic which will be displayed in the section Dynamics.

Parameters:

  • time span for logout: this parameter defines the time span for auto logout in seconds.
  • target frame: this parameter defines the target frame, in which the logout display will be displayed.
  • logout display: this parameter defines the logout display.
