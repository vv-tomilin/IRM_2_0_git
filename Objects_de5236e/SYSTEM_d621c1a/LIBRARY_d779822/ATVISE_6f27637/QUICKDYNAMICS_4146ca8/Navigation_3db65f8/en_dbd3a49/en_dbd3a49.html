Description - Navigation

This Quick Dynamic generates a navigation bar based on the existing display structure..

To use this quick dynamics, draw a rectangle and apply the dynamics to the rectangle. Custom menus also can be passed as a JSON string.

Parameters:

  • id of menu container
  • id of the content frame
  • minimum z-Index
  • custom menu structure as json string
  • visibility after initialization
  • animation speed factor
  • animation type toggling visibility
  • font family
  • font color for active items
  • font color for inactive items
  • font size
  • visibility of navigation elments
  • alignment of the navigation elments
  • background of the navigation elments
  • color of the navigation elments
  • height of the navigation elments in side layouts
  • width of the navigation elements in centered layouts
  • height of navigation element when png symbols are used
  • width height of navigation element when png symbols are used
  • displayed text in the down navigation element or path to png symbol
  • displayed text in the left navigation element or path to png symbol
  • displayed text in the right navigation element or path to png symbol
  • displayed text in the up navigation element or path to png symbol
  • horizontal content position in home element
  • vertical content position in home element
  • background of the home element
  • font color of the home element
  • height of the home element in side layouts
  • width of the home elment in centered layouts
  • text displayed titel in the home element
  • visibility of icons in navigation element
  • visibility of icons in menu items
  • visibility of icons in home element
  • icon for navigation element down
  • icon for navigation element left
  • icon for navigation element right
  • icon for navigation element up
  • icon for navigation menu item open
  • icon for navigation menu item close
  • icon for navigation home element
  • horizontal content position in menu item
  • vertical content position in menu item
  • number of visible menu items (0 ... use height and width properties)
  • background color for active menu items
  • background color for inactive menu items
  • border color of menu items
  • border radius of menu items
  • border width of menu items
  • border style of menu items
  • padding between menu items (also home and navigation elements)
  • height of active menu items in centered layouts
  • height of inactive menu items in centered layouts
  • height of menu items in side layouts
  • width of active menu items in side layouts
  • width of active menu items in side layouts
  • width of active menu items in centered layouts
  • scroll factor for general clients
  • scroll factor for mobile clients
  • scroll factor for panel clients
  • alignment of the navigation
  • takeover of the color and frame definitions from the svg container
  • background color of the menu panel
  • border color of the menu panel
  • border width of the menu panel
  • border style of the menu panel
  • necessary right
  • activation address
  • activation value

Create a custom menu

To create a custom menu, the structure of the menu must be passed as JSON string.

The following is the pattern structure and the example call of the quick dynamics with the parameter for user-defined menu structures.

  	var customMenu = [
		{
			"name": "MY MENU",
			"identifier": "AGENT.DISPLAYS.MAIN",
			"sub": [
				{
					"name": "my display 1",
					"identifier": "AGENT.DISPLAYS.MAIN.my_display_1",
					"display": "AGENT.DISPLAYS.MAIN.my_display_1",
					"width": 45,
					"height": 100
				},
				{
					"name": "my sub menu",
					"identifier": "AGENT.DISPLAYS.MAIN.my_sub_menu",
					"sub": [
					{
						"name": "my display 2",
						"identifier": "AGENT.DISPLAYS.MAIN.my_display_2",
						"display": "AGENT.DISPLAYS.MAIN.my_display_2",
						"width": 45,
						"height": 100
					}
					]
				},
			]
		},
		{
			"name": "SYSTEM",
			"identifier": "AGENT.DISPLAYS.MAIN.system",
			"display": "AGENT.DISPLAYS.MAIN.system",
			"width": 45,
			"height": 100
		}
	];

	webMI.callExtension("SYSTEM.LIBRARY.ATVISE.QUICKDYNAMICS.Navigation", {
		"id": "my_custom_menu",
		"customMenu": JSON.stringify(customMenu)
	});
