Beschreibung - Navigation

Diese Quick Dynamic generiert eine Navigationsleiste auf Basis der vorhandenen Displaystruktur.

Um diese Quick Dynamik zu verwenden, zeichnen Sie ein Rechteck und weisen Sie dem Rechteck die Dynamik zu. Benutzerdefinierten Menüs können ebenfalls als JSON-Zeichenfolge übergeben werden.

Parameters:

  • ID des Menücontainers
  • ID des Inhaltsbereichs
  • Minimaler z-Index
  • Benutzerdefinierte Menüstruktur als JSON-Zeichenfolge
  • Sichtbarkeit nach der Initialisierung
  • Geschwindigkeitsfaktor der Animation
  • Animationstyp, der die Sichtbarkeit umschaltet
  • Schriftfamilie
  • Schriftfarbe für aktive Elemente
  • Schriftfarbe für inaktive Elemente
  • font size
  • Sichtbarkeit der Navigationselemente
  • Ausrichtung der Navigationselemente
  • Hintergrund der Navigationselemente
  • Farbe der Navigationselemente
  • Höhe der Navigationselemente in Seitenlayouts
  • Breite der Navigationselemente in zentrierten Layouts
  • Höhe des Navigationselements bei Verwendung von PNG-Symbolen
  • Breite Höhe des Navigationselements bei Verwendung von PNG-Symbolen
  • Angezeigter Text im Abwärtsnavigationselement oder Pfad zum PNG-Symbol
  • Angezeigter Text im linken Navigationselement oder Pfad zum PNG-Symbol
  • Angezeigter Text im rechten Navigationselement oder Pfad zum PNG-Symbol
  • Angezeigter Text im Navigationselement nach oben oder Pfad zum PNG-Symbol
  • Horizontale Inhaltsposition im Ausgangselement
  • Vertikale Inhaltsposition im Ausgangselement
  • Hintergrund des Home-Elements
  • Schriftfarbe des Home-Elements
  • Höhe des Home-Elements in Seitenlayouts
  • Breite des Hauses in zentrierten Layouts
  • Angezeigter Text Titel im Home-Element
  • Sichtbarkeit von Icons im Navigationselement
  • Sichtbarkeit von Symbolen in Menüpunkten
  • Sichtbarkeit von Symbolen in Menüpunkten
  • Symbol für Navigationselement nach unten
  • Symbol für Navigationselement links
  • Symbol für Navigationselement rechts
  • Symbol für Navigationselement nach oben
  • Symbol für Navigationsmenüpunkt geöffnet
  • Symbol für Navigationsmenüpunkt schließen
  • Symbol für das Navigationselement Startseite
  • Horizontale Inhaltsposition im Menüpunkt
  • Vertikale Inhaltsposition im Menüpunkt
  • Anzahl sichtbarer Menüpunkte (0 ... Höhe und Breite verwenden)
  • Hintergrundfarbe für aktive Menüpunkte
  • Hintergrundfarbe für inaktive Menüpunkte
  • Rahmenfarbe der Menüpunkte
  • Rahmenradius der Menüpunkte
  • Rahmenbreite der Menüpunkte
  • Rahmenart der Menüpunkte
  • Auffüllen zwischen Menüelementen (auch Home- und Navigationselemente)
  • Höhe der aktiven Menüpunkte in zentrierten Layouts
  • Höhe inaktiver Menüpunkte in zentrierten Layouts
  • Höhe der Menüpunkte in Seitenlayouts
  • Breite der aktiven Menüpunkte in Seitenlayouts
  • Breite der aktiven Menüpunkte in Seitenlayouts
  • Breite der aktiven Menüpunkte in zentrierten Layouts
  • Scrollfaktor für allgemeine Clients
  • Scrollfaktor für mobile Clients
  • Scrollfaktor für Panel-Clients
  • Ausrichtung der Navigation
  • übernahme der Farb- und Rahmendefinitionen aus dem SVG-Container
  • Hintergrundfarbe der Menüleiste
  • Rahmenfarbe der Menüleiste
  • Rahmenbreite des Menüfeldes
  • Rahmenstil der Menüleiste
  • Benötigtes Recht: Für die Benutzung muss der Benutzer über dieses Recht verfügen.
  • Aktivierungsadresse: Adresse für den Aktivierungswert.
  • Aktivierungswert: Wert für die Aktivierung.

Benutzerdefiniertes Menu erstellen

Um ein benutzerdefiniertes Menü zu erstellen, muss die Struktur des Menüs als JSON-Zeichenfolge übergeben werden.

Nachfolgend ist die Musterstruktur und der Beispielaufruf der Schnelldynamik mit dem Parameter für benutzerdefinierte Menüstrukturen.

  	var customMenu = [
		{
			"name": "MY MENU",
			"identifier": "AGENT.DISPLAYS.MAIN",
			"sub": [
				{
					"name": "my display 1",
					"identifier": "AGENT.DISPLAYS.MAIN.my_display_1",
					"display": "AGENT.DISPLAYS.MAIN.my_display_1",
					"width": 45,
					"height": 100
				},
				{
					"name": "my sub menu",
					"identifier": "AGENT.DISPLAYS.MAIN.my_sub_menu",
					"sub": [
					{
						"name": "my display 2",
						"identifier": "AGENT.DISPLAYS.MAIN.my_display_2",
						"display": "AGENT.DISPLAYS.MAIN.my_display_2",
						"width": 45,
						"height": 100
					}
					]
				},
			]
		},
		{
			"name": "SYSTEM",
			"identifier": "AGENT.DISPLAYS.MAIN.system",
			"display": "AGENT.DISPLAYS.MAIN.system",
			"width": 45,
			"height": 100
		}
	];

	webMI.callExtension("SYSTEM.LIBRARY.ATVISE.QUICKDYNAMICS.Navigation", {
		"id": "my_custom_menu",
		"customMenu": JSON.stringify(customMenu)
	});
