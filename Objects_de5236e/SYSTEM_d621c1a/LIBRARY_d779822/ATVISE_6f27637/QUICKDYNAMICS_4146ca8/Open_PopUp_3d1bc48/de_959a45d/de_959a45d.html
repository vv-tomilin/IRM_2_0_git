Beschreibung - Fenster einblenden

Dieses Quick Dynamic öffnet ein Display als eingeblendetes Fenster in Abhängigkeit von einem der mit "onEvent" definierten Ereignisse.

Parameter:

  • Popup: Display, das als eingeblendetes Fenster geöffnet wird
  • onEvent: Das Ereignis, das dieses Quick Dynamic aktiviert, wobei es sich um eines der folgenden handelt:
    • click: Mausklick
    • dblclick: Doppelklicken mit der Maustaste
    • mousedown: Drücken der Maustaste
    • mouseup: Loslassen der Maustaste
    • mouseover: Bewegen des Mauszeigers über das angewendete grafische Element
    • mousemove: Bewegen der Maus an beliebiger Stelle über das angewendete grafische Element
    • mouseout: Bewegen des Mauszeigers aus dem angewendeten grafischen Element heraus
  • width: Breite des eingeblendeten Fensters (in Pixeln)
  • height: Höhe des eingeblendeten Fensters (in Pixeln)
  • extern: Wenn wahr, wird das eingeblendete Fenster im Fenster eines externen Webbrowsers geöffnet.
  • menubar: Wenn wahr, wird eine Menüleiste angezeigt.
  • moveable: Wenn wahr, kann das eingeblendete Fenster verschoben werden.
  • resizeable: Wenn wahr, kann die Größe des eingeblendeten Fensters geändert werden.
  • scrollbars: Wenn wahr, kann das eingeblendete Fenster Bildlaufleisten haben.
  • status: Wenn wahr, wird eine Statusleiste angezeigt.
  • title: Legt den Titel des eingeblendeten Fensters fest.
  • toolbar: Wenn wahr, wird eine Werkzeugleiste angezeigt.
  • modal: Wenn wahr, wird ein eingeblendetes Fenster modal geöffnet, d. h. immer im Vordergrund.
  • alle Parameter übergeben: Wenn wahr, werden alle Parameter übergeben.
