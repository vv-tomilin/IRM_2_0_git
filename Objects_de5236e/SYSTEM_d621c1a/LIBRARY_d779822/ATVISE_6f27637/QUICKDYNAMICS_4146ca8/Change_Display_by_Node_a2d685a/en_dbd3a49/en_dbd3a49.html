Description - Change Display by Node

This Quick Dynamic opens a new display if the value of the node defined by "nodeID" lies within the range defined by "minRange" and "maxRange".

Parameters:

  • nodeID: node triggering this Quick Dynamic
  • minRange: lower bound of the range in which the node's value should lie
  • maxRange: upper bound of the range in which the node's value should lie; if not specified, it will be set to "minRange", i.e. the quick dynamic will be triggered at a
    discrete value (minRange=maxRange=value) only, and not for a range
  • Display: the display which is to be opened
