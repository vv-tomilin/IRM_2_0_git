Beschreibung - Text festlegen

Dieses Quick Dynamic legt den Text des angewendeten Textelements auf den Wert des definierten Datenpunkts mit der definierten Anzahl Dezimalstellen (max. 9) fest.

Parameter:

  • nodeID: Datenpunkt, dem der Wert entnommen wird (Variablen müssen vom Typ "Zahl", "Boole" oder "Strings" sein; Strings dürfen nur Ziffern enthalten)
  • decimalPlaces: Anzahl der für das Anzeige zu verwendenden Dezimalstellen (nicht mehr als 9)
