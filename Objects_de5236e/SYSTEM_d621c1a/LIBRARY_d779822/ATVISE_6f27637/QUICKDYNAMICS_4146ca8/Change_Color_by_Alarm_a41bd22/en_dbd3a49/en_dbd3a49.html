Description - Change Color by Alarm

This Quick Dynamic changes the color and stroke of the applied graphic element depending on the different alarm states.

Parameters:

  • alarm: alarm (node) which will trigger this Quick Dynamic
  • alarmOnFill: color to fill the graphic element if an alarm occurs
  • alarmOnStroke: color for the stroke of the graphic element if an alarm occurs
  • alarmOffFill: fill color of the graphic element to set it back to the original state when the alarm is off or acknowledged
  • alarmOffStroke: stroke color of the graphic element to set it back to the original state when the alarm is off or acknowledged
  • blinkOnUnacknowledged: flash interval for alarm state "on, unacknowledged"
  • blinkOnAcknowledged: flash interval for alarm state "on, acknowledged"
  • blinkOffUnacknowledged: flash interval for alarm state "off, unacknowledged"
  • blinkOnOffUnacknowledged: flash interval for alarm state "on off, unacknowledged"
Flash interval for alarm state:
  • visible
  • invisible
  • Flash every 250ms
  • Flash every 500ms
  • Flash every 750ms
  • Flash every 1500ms
