Description - Auto Reconnect

This Quick Dynamic enables the auto reconnect functionality and provides an additional keepalive mechanism.

Please note: The auto reconnect functionality is activated by default with a 5 seconds interval; the keepalive mechanism is activated too. To change the default behaviour add the Quick Dynamic to the Default Display and adjust the parameters.

Parameters:

  • activates auto reconnect: use this parameter for enabling or disabling the auto reconnect functionality. (default: true)
  • interval in seconds for auto reconnect: this parameter defines the time span for auto reconnect in seconds. (default: 5)

Note:
The parameter for the interval in seconds for verifying if server is alive (keepalive-mechanism) is omitted in this Quick Dynamic. The keepalive-interval can be configured in the webmicfg.js - check the manual for further details.
