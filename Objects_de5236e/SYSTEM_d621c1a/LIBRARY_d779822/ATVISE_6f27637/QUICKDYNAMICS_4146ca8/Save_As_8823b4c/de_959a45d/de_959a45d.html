Beschreibung - Speichern unter

Dieses Quick Dynamic kann verwendet werden um einen Datei Download im Browser zu triggern.
Hinweis: Funktioniert nur mit Servern die Serverscripts unterstützen. Wird das angegebene Serverscript nicht gefunden, wird stattdessen ein Popup mit dem Dateiinhalt angezeigt.
Der einzige Parameter der notwendig ist für dieses Quick Dynamic ist "content", alle anderen Parameter können wie gewünscht modifiziert werden.
Hinweis: Da nur ecmascript Zeichenketten als "content" verwendet werden können, ist der "type" auf text beschränkt. Alle anderen Typen (wie zum Beispiel binäre Daten) werden nicht unterstützt.

Parameter:

  • name: Name der Datei (der Standard ist "filename.txt")
  • type: Mimetype der Datei (der Standard ist "text/plain")
  • content: Inhalt der Datei
  • action: url action oder Serverscript das ausgeführt wird (der Standard ist "/export")
  • method: post oder get Methode (der Standard ist "post")
  • enctype: Encoding Typ (der Standard ist "application/x-www-form-urlencoded")
  • useBOM: Export ohne oder mit "Byte Order Mark" (stellt MS Excel Kompatibilität für Sonderzeichen sicher)
