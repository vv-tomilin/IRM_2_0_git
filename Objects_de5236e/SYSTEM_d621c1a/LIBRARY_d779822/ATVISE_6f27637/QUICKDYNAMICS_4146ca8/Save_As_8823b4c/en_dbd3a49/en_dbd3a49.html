Description - Save As

This Quick Dynamic can be used to trigger a file download in the browser.
Note: This only works with servers that support serverscripts. In case the serverscript is unavailable, a popup will be opened with the given content instead.
The only required parameter for this Quick Dynamic is "content", all other parameters can be modified as needed.
Note: Since we can only use ecmascript strings as "content", the "type" parameter is limited to text-only. All other types (like binary data) are not supported.

Parameters:

  • name: name of the file (the default is "filename.txt")
  • type: mimetype of the file (the default is "text/plain")
  • content: content of the file
  • action: url action or serverscript that will be executed (the default is "/export")
  • method: post or get method (the default is "post")
  • enctype: encoding type (the default is "application/x-www-form-urlencoded")
  • useBOM: export with or without "byte-order-mark" (ensures that special characters are displayed correctly in MS Excel)
