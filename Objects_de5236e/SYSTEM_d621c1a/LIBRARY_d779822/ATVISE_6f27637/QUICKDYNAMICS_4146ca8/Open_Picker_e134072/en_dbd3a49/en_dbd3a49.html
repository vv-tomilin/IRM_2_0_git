Description - Open Popup

This Quick Dynamic opens a date or color picker display as a popup.

Parameters:

  • picker dialog address: the node address of the picker dialog.
  • picker dialog width: the width of the picker dialog.
  • picker dialog height: the height of the picker dialog.
  • popup title: the title shown in the header bar of the pop up window.
  • parameters: additional parameters given to the picker dialog (e.g. webMI.query parameters of the parent object display).
  • style: style parameters as defined in webMI.openWindow.
  • open centered: open the picker dialog centered on the working screen or attached to the calling object display.
  • calling element width: width of the calling object display.
  • calling element height: height of the calling object display.
