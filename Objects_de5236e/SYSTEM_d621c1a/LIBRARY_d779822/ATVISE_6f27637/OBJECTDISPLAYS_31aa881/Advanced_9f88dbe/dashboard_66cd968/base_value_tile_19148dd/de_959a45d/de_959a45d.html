Beschreibung

Das Display stellt den aktuellen Wert einer Prozessgröße dar.

Mit Hilfe einer Statusanzeige in Form eines zusätzlichen Rahmens kann zudem auch ein negativer Status des aktuellen Wertes und ein zusätzlich definierter Alarm dargestellt werden. Die Farben und Blinkintervalle werden in globalen Listen definiert.

Parameter

  • base: Adresse zur Ausgabe- und Statusüberwachung..
  • Alarmadresse: Optionale Adresse für die Alarmüberwachung.

Ausgabeeinstellung

  • Titel: Titel der angezeigten Kachel.
  • Einheit: Diese Einheit wird zusätzlich zum Wert angezeigt.
  • Format: Ausgabeformatierung des Wertes (Standard: string).
  • Nachkommastellen: Anzahl der Stellen rechts vom Dezimalkomma (Standard: 0).
  • Schriftart Titel: Schriftfamilie, die auf den Titel angewendet wird.
  • Schriftart Ausgabe: Schriftfamilie, die auf die Ausgabe (Wert/Einheit) angewendet wird.
  • Schriftgröße Titel: Schriftgröße, die auf den Titel angewendet wird.
  • Schriftgröße Wert: Schriftgröße für Wert bei mehrzeiliger Ausgabe (Standard: 20).
  • Schriftgröße Einheit: Schriftgröße für Einheit bei mehrzeiliger Ausgabe (Standard: 20).
  • Schriftgröße einzeilig: Schriftgröße für Wert und Einheit bei einzeiliger Ausgabe (Standard: 20).
  • Einzeilig: Ein- oder mehrzeilige Darstellung (Standard: false).

Verhalten bei Klick:

  • Klickereignis aktivieren: Aktivierung der Bearbeitung eines Klicks auf die Kachel.
  • Klick auf Inhalt erlauben: Zusätzliche Erkennung eines Klicks auf den Kachelinhalt.
  • Zielanzeige: Anzeige, die beim Erkennen eines Klicks geöffnet werden soll.
  • Verknüpftes Element: Element, das auf externe Klickereignisse überwacht werden soll (ID des Elements).

Status:

  • Statusüberwachung aktivieren: Schaltet die Anzeige des aktuellen Wertestatus ein oder aus.
  • Statustrigger feuern: Statt der Statusanzeige kann auch ein hier definierter Trigger global gefeuert werden.

Darstellung:

  • Hintergrundfarbe: Hintergrundfarbe (Standard: Globaler Parameter atvFillColor)
  • Schriftfarbe: Schriftfarbe (Standard: Globaler Parameter atvFontColor)
  • Rahmenfarbe: Rahmenfarbe (Standard: Globaler Parameter atvBorderColor)
  • Deaktivierungsfarbe: Deaktivierungsfarbe (Standard: Globaler Parameter atvInactiveColor)
  • Statushinweis: Spezifiziert die Darstellung des Status über eine definierte globale Liste (Standard: Globaler Parameter atvStatusIndication)
  • Alarmhinweis: Spezifiziert die Darstellung des Alarms überber eine definierte globale Liste (Standard: Globaler Parameter atvAlarmIndication)

Optionen:

  • Tooltip: Zeigt eine Tooltip Box mit dem eingegebenen Text an (Standard HTML-Elemente können auch eingegeben werden)

Sicherheit:

  • Benötigtes Recht: Der Benutzer muss über dieses Recht verfügen, um diese Schaltfläche zu aktivieren, d. h. eine der beiden Schaltflächen betätigen zu können.
  • Aktivierungsadresse: Stimmt der Wert dieser Adresse mit dem "Aktivierungswert" überein, ist das Steuerelement aktiv, so dass es betätigt werden kann.
  • Aktivierungswert: Wert, von dem das Steuerelement aktiviert wird
