Beschreibung

Dieses Element zeigt einen beliebigen Text in einem HTML Container (ForeignObject) an.

Parameter

    • base: Base Adresse des Displays.
    • Text: Text welcher angezeigt werden soll.

  2. Parametergruppe: Schrift
    • Schriftart: Schriftart, die auf den Wert angewendet wird (Standard: Arial).
    • Größe: Schriftgröße, die auf den Wert angewendet wird (Standard: 12).
    • Schriftstärke: Schriftstärke, die auf den Wert angewendet wird (Standard: normal).
    • Farbe: Farbe, die auf den Wert angewendet wird (Standard: Globaler Parameter atvFontColor)

  3. Parametergruppe: Ausrichtung
    • horizontal: Horizontale Ausrichtung des Textes (Standard: left)
    • vertikal: Vertikale Ausrichtung des Textes (Standard: top)
