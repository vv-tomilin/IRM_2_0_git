Beschreibung

Dieses Display kann als Dialogfeld zur Generierung einer Farbe im Hexadezimalformat verwendet werden.

Farben können auf unterschiedliche Weise generiert werden, durch Klicken auf die Grundfarben im Kreis oben, durch manuelles Einstellen der RGB/HSV-Werte oder einfach durch Eingeben einer Farbe im Hexadezimalformat. Die aktuelle Farbe wird oben im inneren Kreis angezeigt. Die ausgewählte Standardfarbe kann über den Parameter "Farbe" definiert werden.

Um den Wert der Farbe im Display abzurufen, das dieses Dialogfeld geöffnet hat, muss der Trigger "valuechanged_" mit dem Namensparameter verbunden werden (z. B. "valuechanged_myColorPicker"). In der Regel wird dieses Display als Dialogfeld verwendet, Weitere Informationen finden Sie unter dem Display "Eingabeelement Farbe".

Parameter

Es gibt vier Parameter:

  • Name: eindeutiger Name zur Identifizierung dieses Displays (mandatorisch)
  • Farbe: anfangs angezeigte Farbe
  • Textfarbe: Farbe, die auf die Beschriftung angewendet wird (Standard: Globaler Parameter atvFontColor)
  • Rahmenfarbe: Farbe, die auf Rahmen angewendet wird (Standard: Globaler Parameter atvBorderColor)

Beispiele

Erstellen Sie ein Display mit einer Schaltfläche, die das Display dieses Dialogfelds mit dem Namen "myColorPicker" öffnet und verbinden Sie den Trigger "valuechanged_myColorPicker" für Wertänderungen.
