Beschreibung

Dieses Display zeigt einen definierbaren Dialog für Nachrichten, Informationen oder Warnungen an, welcher beispielsweise über die Funktion openWindow aufgerufen werden kann. Es können frei wählbare Texte in der Kopfzeile und für die Schaltflächen verwendet werden.
Nachrichten können über Trigger an das Display übermittelt werden. Den beiden Schaltflächen können zusätzlich Trigger-Funktionen übergeben werden. Wenn keine Triggernamen übergeben werden, wird als Standardfunktion für die beiden Schaltflächen das Fenster geschlossen.

Anforderungen

Dieses Display erfordert jQuery, welches als Teil der Tabelle Bibliothek mitgeliefert wird.

Um Kompatibilitätsprobleme zu vermeiden, wird die mitgelieferte jQuery Version nur verwendet, wenn keine bereits vorhandene Version erkannt wird. Sollte daher in einem bestehenden Projekt jQuery bereits verwendet werden, greift die Tabellenbibliothek auf diese zurück. Gehen sie daher sicher, dass in diesem Falle eine jQuery Version > 1.7.x verwendet wird!

Parameter

Es gibt insgesamt 9 Parameter:

  • Symbol: Adresse des anzuzeigenden Symbols.
  • Textfarbe: Farbe, die auf die Beschriftung angewendet wird (Standard: Globaler Parameter atvFontColor)
  • Symbolfarbe: Farbe des zu verwendenden Symbols (Standard: Globaler Parameter atvSymbolColor)
  • Text der Kopfzeile: Text in der Kopfzeile des Dialogs.
  • Text Schaltfläche 1: Text der Schaltfläche 1
  • Text Schaltfläche 2: Text der Schaltfläche 2
  • Aktion Schaltfläche 1: Trigger-Funktion der Schaltfläche 1
  • Aktion Schaltfläche 2: Trigger-Funktion der Schaltfläche 2
  • Aktion bei Unload: Trigger-Funktion beim Schließen mit [X]

Trigger

  • com.atvise.message_board_update: Fügt eine Nachricht zum Board
  • com.atvise.message_board_reset: Löscht den Inhalt des Boards

    • Beispiel

    Aufruf des Dialogs mit Hilfe der openWindow Funktion:

    webMI.display.openWindow({display:"SYSTEM.LIBRARY.ATVISE.OBJECTDISPLAYS.Advanced.dialogs.message_board", x:0, y:0, height:600, width:800, modal:true, resizable:false, movable:true, scrollbars:false, menubar:false, status:false, toolbar:false, query: {headline:"Meine Überschrift"}});

    Beispiel

    Hinzufügen einer Nachricht:

    webMI.trigger.fire("com.atvise.message_board_update", ["Nachricht Zeile 1", "Nachricht Zeile 2", ..., "Nachricht Zeile n]);
