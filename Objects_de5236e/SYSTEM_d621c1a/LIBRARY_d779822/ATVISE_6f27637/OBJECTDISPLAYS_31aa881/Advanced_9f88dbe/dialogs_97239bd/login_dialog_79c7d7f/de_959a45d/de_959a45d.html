Beschreibung

Dieses Steuerelement kann nur in Verbindung mit den Steuerelementen "Schaltfläche/Anzeige Login" und ""Schaltfläche Login", nicht aber eigenständig verwendet werden.

Parameter

Es stehen folgende Darstellungs-Parameter zur Verfügung:

  • Textfarbe: Farbe, die auf die Beschriftung angewendet wird (Standard: Globaler Parameter atvFontColor)
  • Füllfarbe: Füllfarbe von Hintergrundflächen (Standard: Globaler Parameter atvFillColor)
  • Symbolfarbe: Farbe des zu verwendenden Symbols (Standard: Globaler Parameter atvSymbolColor)
