Description

This control is only be used by the controls "Button/Display Login" and "Button Login" and shall not be used as standalone control.

Parameters

This control has the following appearance parameters:

  • font color: color which will be applied to the labels (default: global parameter atvFontColor)
  • fill color: background fill color (default: global parameter atvFillColor)
  • symbol color: color of the symbol to be used (default: global parameter atvSymbolColor)
