Description

The alarmlog is a control that displays the last 3 alarm events that occured in the system.
All states of the alarm (on unacknowledged » on acknowleged » off acknowledged OR on unacknowledged » off unacknowledged) are displayed with different symbols.
Additionally the address (source node) and alarmtext (from alarm configuration) are displayed.

Parameters

  • Font
    • family: font family
    • size: font size
    • color: font color (default: global parameter atvFontColor)
  • Alarm
    • min alarm priority: minimum alarm priority to be displayed
    • max alarm priority: maximum alarm priority to be displayed
  • Appearance
    • symbol color: fill color of the alarm symbols (default: global parameter atvSymbolColor)
