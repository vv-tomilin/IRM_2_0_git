Beschreibung

"area" ist ein einfaches Steuerelement, das als Hintergrundgrafik mit einem Titeltext verwendet werden kann.

Parameter

  • Füllung: Füllfarbe für den Hintergrund (Standard: Globaler Parameter atvFillColor)
  • Kontur: Konturfarbe des Elements (Standard: Globaler Parameter atvBorderColor)
  • Schriftfarbe: Farbe der Schrift (Standard: Globaler Parameter atvFontColor)
  • Bereichsbeschreibung: Beschriftung des Elements
