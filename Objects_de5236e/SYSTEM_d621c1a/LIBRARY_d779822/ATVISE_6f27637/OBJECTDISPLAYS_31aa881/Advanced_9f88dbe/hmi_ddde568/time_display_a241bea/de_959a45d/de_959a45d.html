Beschreibung

Dieses Steuerelement zeigt die aktuelle Client-Zeit an (Stunden und Minuten).

Parameter

  • base: Der Wert dieser Adresse wird gesetzt.
  • Schriftfarbe: Schriftfarbe, die auf die Werte angewendet wird (Standard: Globaler Parameter atvFontColor)
  • Füllfarbe: Hintergrundfarbe des Steuerelements (Standard: Globaler Parameter atvFillColor)
  • Rahmenfarbe: Farbe der Umrahmung des Steuerelements (Standard: Globaler Parameter atvBorderColor)
