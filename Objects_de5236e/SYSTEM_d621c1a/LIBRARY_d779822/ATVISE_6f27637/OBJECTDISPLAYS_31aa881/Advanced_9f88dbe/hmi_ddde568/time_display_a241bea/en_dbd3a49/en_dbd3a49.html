Description

This control displays the current client time (hour and minutes).

Parameters

  • base: the value of this address will be set
  • font color: font color which will be applied to the values (default: global parameter atvFontColor)
  • fill color: background color of the control (default: global parameter atvFillColor)
  • border color: color of the control's border (default: global parameter atvBorderColor)
