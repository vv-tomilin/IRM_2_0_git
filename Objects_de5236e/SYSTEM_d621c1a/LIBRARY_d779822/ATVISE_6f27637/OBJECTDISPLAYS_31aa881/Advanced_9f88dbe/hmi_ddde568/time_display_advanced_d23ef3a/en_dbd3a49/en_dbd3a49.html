Description

This control displays the current client time (hours, minutes, seconds and milliseconds).

Parameters

  • base: if there is no base specified then the current client time will be displayed.
  • update time interval: if there is no base specified then it is possible to enter an update time interval (in ms) which specifies the interval in which the current client time will be updated.
  • font color: font color which will be applied to the values (default: global parameter atvFontColor)
  • fill color: background color of the control (default: global parameter atvFillColor)
  • border color: color of the control's border (default: global parameter atvBorderColor)
