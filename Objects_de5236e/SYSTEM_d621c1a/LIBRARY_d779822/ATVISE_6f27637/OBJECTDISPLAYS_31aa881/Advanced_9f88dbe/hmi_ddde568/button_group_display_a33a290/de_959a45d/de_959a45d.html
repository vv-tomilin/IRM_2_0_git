Beschreibung

Bei diesem Steuerelement handelt es sich um eine Gruppe von Schaltflächen, die alle Displays aus der Display-Hauptstruktur anzeigen, wenn der Mauszeiger auf sie gerichtet wird:

  • MAIN: Alle Displays im Ordner "DISPLAYS.MAIN"
  • USER: Alle Displays im Ordner "DISPLAYS.USER"
  • SYSTEM: Alle Displays im Ordner "DISPLAYS.SYSTEM"
  • REPORT: Alle Displays im Ordner "DISPLAYS.REPORT"

Parameter

  • Zielframe: Name des IFrames, in dem das Display geöffnet wird.
  • Darstellung
    • Füllfarbe: Button- und Hintergrundfarbe (Standard: Globaler Parameter atvFillColor)
    • Rahmenfarbe: Farbe des Rahmens um das Steuerelement (Standard: Globaler Parameter atvBorderColor)
    • Symbolfarbe: Füllfarbe für Symbole (Standard: Globaler Parameter atvSymbolColor)
    • Schriftfarbe: Schriftfarbe, die auf Main-Menus angewendet wird, welche sich bei Klick auf Schaltfläche öfnen (Standard: Globaler Parameter atvFontColor)
