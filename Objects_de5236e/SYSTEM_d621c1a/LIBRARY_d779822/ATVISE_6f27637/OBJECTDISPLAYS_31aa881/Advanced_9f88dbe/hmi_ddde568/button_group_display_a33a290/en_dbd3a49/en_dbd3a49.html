Description

This control is a group of buttons that shows all displays from the main display structure on mouseover:

  • MAIN: All displays that are located in the "DISPLAYS.MAIN" folder.
  • USER: All displays that are located in the "DISPLAYS.USER" folder.
  • SYSTEM: All displays that are located in the "DISPLAYS.SYSTEM" folder.
  • REPORT: All displays that are located in the "DISPLAYS.REPORT" folder.

Parameters

  • target frame: name of the iframe in which the display will be openend
  • Appearance
    • fill color: color of button and background (default: global Parameter atvFillColor)
    • border color: color of the control's border (default: global parameter atvBorderColor)
    • symbol color: fill color for the symbols (default: global parameter atvSymbolColor)
    • color: font color, applied to main menus which open on button click (default: global parameter atvFontColor)
