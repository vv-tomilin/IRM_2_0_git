Description

This control switches the different languages that are defined in the system and displays the currently selected language or an individual text defined by a parameter value.

Parameters

  • language text: if the value of this parameter is empty then the currently selected language will be displayed, otherwise the value will be displayed.
  • Font
    • family: font family
    • size: font size
    • color: font color (default: global parameter atvFontColor)
  • Appearance
    • symbol color: color of the symbol (default: global parameter atvSymbolColor)
    • fill color: background color of the control (default: global parameter atvFillColor)
    • border color: color of the button's border (default: global parameter atvBorderColor)
    • stroke color when pressed: color of the border when pressing the button
