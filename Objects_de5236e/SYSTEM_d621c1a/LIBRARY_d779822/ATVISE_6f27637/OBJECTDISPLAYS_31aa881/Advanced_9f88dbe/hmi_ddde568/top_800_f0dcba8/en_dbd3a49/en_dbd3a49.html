Description

This control is a small version of the default "top" display (e.g. for panel screens).
It's a collection of the most important controls (display main menu, alarm handling, login and language buttons...).

Parameters

  • Font
    • font color: font color which will be applied to the values (default: global parameter atvFontColor)
    • background color: the control's background color (default: global parameter atvFillColor)
  • Alarm
    • min alarm priority: minimum alarm priority to be displayed
    • max alarm priority: maximum alarm priority to be displayed
