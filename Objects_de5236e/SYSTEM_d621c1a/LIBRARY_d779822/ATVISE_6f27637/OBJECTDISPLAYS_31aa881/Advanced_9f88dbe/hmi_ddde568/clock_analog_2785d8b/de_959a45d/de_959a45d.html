Beschreibung

Dieses Steuerelement zeigt die aktuelle Client-Zeit an.

Parameter

  • Textfarbe: Farbe, die auf Wert, Einheit und Beschriftung angewendet wird (Standard: Globaler Parameter atvFontColor)
  • Füllfarbe: Füllfarbe des Uhrrahmens (Standard: Globaler Parameter atvFillColor)
  • Rahmencolor: Farbe der Uhrumrahmung (Standard: Globaler Parameter atvBorderColor)
  • Sekundenzeiger Farbe: Farbe des Sekundenzeigers der Uhr (Standard: Globaler Parameter atvAccent1)
  • Symbolfarbe: Farbe der Symbole der Uhrskala (Standard: Globaler Parameter atvSymbolColor)
