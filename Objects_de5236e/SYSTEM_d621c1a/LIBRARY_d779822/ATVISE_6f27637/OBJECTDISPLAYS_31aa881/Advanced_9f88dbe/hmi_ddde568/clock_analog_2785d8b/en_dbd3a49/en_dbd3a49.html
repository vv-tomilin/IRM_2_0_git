Description

This control displays the current client time.

Parameters

  • font color: color which will be applied to the clock's labeling (default: global parameter atvFontColor)
  • fill color: fill color of the clock frame (default: global parameter atvFillColor)
  • border color: color of the button's border (default: global parameter atvBorderColor)
  • second-pointer color: color of the clock's second-pointer (default: global parameter atvAccent1)
  • symbol color: color of the clock's scale symbols (default: global parameter atvSymbolColor)
