Beschreibung

Dieses Steuerelement ist eine kleinere Version des Standard-Displays "Kopf".
Es handelt sich um eine Gruppe der wichtigsten Steuerelemente (Anzeige des Hauptmenüs, Alarmbehandlung, Schaltflächen für Login und Sprache...).

Parameter

  • Schriftfarbe: Schriftfarbe, die auf die Werte angewendet wird (Standard: Globaler Parameter atvFontColor)
  • Hintergrundfarbe: Hintergrundfarbe des Steuerelements (Standard: Globaler Parameter atvFillColor)
