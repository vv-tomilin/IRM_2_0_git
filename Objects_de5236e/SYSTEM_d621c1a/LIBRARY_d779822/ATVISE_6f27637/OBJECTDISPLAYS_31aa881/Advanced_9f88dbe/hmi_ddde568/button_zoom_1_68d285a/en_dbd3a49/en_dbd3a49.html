Description

This button resets the zoom level of the visualisation.

Parameters

  • target frame name: name of the iframe display that shall be zoomed
  • Appearance
    • symbol: the symbol of the button
    • symbol pressed: the symbol that will displayed when the button is pressed
    • symbol color: color of the symbol (default: global parameter atvSymbolColor)
    • symbol color pressed: color of the symbol in pressed state (default: global parameter atvSymbolColor)
    • fill color: background color of the control (default: global parameter atvFillColor)
    • border color: color of the button's border (default: global parameter atvBorderColor)
    • stroke color pressed: color of the border when pressing the button
