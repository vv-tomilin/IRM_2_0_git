Description

This control is a minimal version of the default "top" display (e.g. for panel screens).
It's a collection of important controls (display main menu, login and language button and display of the logged in user).

Parameters

  • font color: font color which will be applied to the values (default: global parameter atvFontColor)
  • background color: the control's background color (default: global parameter atvFillColor)
