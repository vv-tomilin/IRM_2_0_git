Description

This control displays the currently logged in user or a default text if no user is logged in.

Parameters

  • login text: text that will be displayed if no user is logged in.
  • Font

    • family: font family
    • size: font size
    • color: font color (default: global parameter atvFontColor)

    Appearance

    • fill color: fill color of the control (default: global parameter atvFillColor)
    • value field fill color: background color of the value field
    • border color: color of the control's border (default: global parameter atvBorderColor)
