Beschreibung

Über dieses Steuerelement wird der derzeit eingeloggte Benutzer angezeigt oder ein Standardtext, wenn kein Benutzer eingeloggt ist.

Parameter

  • Logintext: Text, der angezeigt wird, wenn kein Benutzer eingeloggt ist
  • Schrift

    • Schriftart: Schriftart
    • Größe: Schriftgröße
    • Farbe: Schriftfarbe, die auf die Werte angewendet wird (Standard: Globaler Parameter atvFontColor)

    Appearance

    • Füllfarbe: Hintergrundfarbe des Werts (Standard: Globaler Parameter atvFillColor)
    • Füllfarbe Wertefeld: Füllfarbe des Wertefelds (Standard: Globaler Parameter atvFillColor)
    • Rahmenfarbe: Farbe des Rahmens um das Steuerelement (Standard: Globaler Parameter atvBorderColor)
