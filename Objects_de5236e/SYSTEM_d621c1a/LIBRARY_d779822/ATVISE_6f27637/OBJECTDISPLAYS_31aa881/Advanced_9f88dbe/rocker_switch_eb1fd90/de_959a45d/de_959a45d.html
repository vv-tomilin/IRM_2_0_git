Beschreibung

Über diese Schaltfläche wird der Wert einer Adresse erhöht oder verringert, wie mit der Schaltfläche "Erhöhen/Verringern".

Darüber hinaus wird der Trigger "valuechanged" mit dem neuen Wert (= erhöhter oder verringerter Wert) aktiviert.

Mit Hilfe einer Statusanzeige in Form eines zusätzlichen Rahmens kann zudem auch ein negativer Status des aktuellen Wertes und ein zusätzlich definierter Alarm dargestellt werden. Die Farben und Blinkintervalle werden in globalen Listen definiert.

Parameter

Dieses Steuerelement hat die folgenden allgemeinen Parameter:

  • base: Der Wert dieser Adresse wird entweder erhöht oder verringert.
  • Alarmadresse: Optional zusätzliche Adresse eines Alarms, welcher angezeigt werden soll.
  • Schrittweite zum Erhöhen: Wert, um den der Wert von "base" erhöht wird (Standard: 1)
  • Schrittweite zum Verringern: Wert, um den der Wert von "base" verringert wird (Standard: 1)
  • Intervall [ms] (0 = aus): Legt fest, wie lange die Schaltfläche betätigt werden muss, damit der Wert von "base" automatisch geändert wird (Standard: 150 ms, d. h., wird die Schaltfläche kontinuierlich betätigt, wird der Wert alle 150 ms geändert).
  • Statusüberwachung aktivieren: Schaltet die Anzeige des aktuellen Wertestatus ein oder aus.
  • Statustrigger feuern: Statt der Statusanzeige kann auch ein hier definierter Trigger global gefeuert werden.
Schrift:
  • Schriftart: Schriftart, die auf den Wert angewendet wird (Standard: Arial)
  • Größe: Schriftgröße, die auf den Wert angewendet wird (Standard: 12)
  • Farbe:Farbe, die auf den Wert angewendet wird (Standard: Globaler Parameter atvFontColor)
Darstellung:
  • Füllfarbe Schaltfläche: Füllfarbe von Hintergrundflächen (Standard: Globaler Parameter atvFillColor)
  • Füllfarbe Wertefeld: Hintergrundfarbe des Wertefelds (Standard: weiß)
  • Füllfarbe Wertefeld inaktiv: Füllfarbe des Wertefelds, wenn nicht aktiv (Standard: dunkelgrau)
  • Füllfarbe Pfeil: Füllfarbe des Pfeils (Standard: Globaler Parameter atvSymbolColor)
  • Konturfarbe Schaltfläche gedrückt: Farbe der Kontur, die bei Betätigung einer der beiden Schaltflächen eingestellt wird (Standard: weiß)
  • Fokus Konturfarbe: Farbe der Kontur, die beim Fokus der Schaltfläche eingestellt wird (Standard: weiß)
  • Rahmenfarbe: Farbe des Rahmens um das Steuerelement (Standard: Globaler Parameter atvBorderColor)
  • Statushinweis: Spezifiziert die Darstellung des Status über eine definierte globale Liste (Standard: Globaler Parameter atvStatusIndication)
  • Alarmhinweis: Spezifiziert die Darstellung des Alarms überber eine definierte globale Liste (Standard: Globaler Parameter atvAlarmIndication)
Optionen:
  • Minimum: Legt das untere Limit für den zu ändernden Wert fest, d. h., wenn der aktuelle Wert mit diesem Minimum übereinstimmt, wird er nicht weiter verringert (Standard: kein Minimum festgelegt).
  • Maximum: Legt das obere Limit für den zu ändernden Wert fest, d. h., wenn der aktuelle Wert mit diesem Maximum übereinstimmt, wird er nicht weiter erhöht (Standard: kein Maximum festgelegt).
  • Konsistenzgruppe: Name der Konsistenzgruppe.
  • Tab-Index: Bestimmt die Reihenfolge, in der mit der Tabulatortaste zwischen den einzelnen Object Displays gewechselt werden kann
  • Tooltip: Zeigt eine Tooltip Box mit dem eingegebenen Text an (Standard HTML-Elemente können auch eingegeben werden)
Sicherheit:
  • Benötigtes Recht: Der Benutzer muss über dieses Recht verfügen, um diese Schaltfläche zu aktivieren, d. h. eine der beiden Schaltflächen betätigen zu können.
  • Aktivierungsadresse: Stimmt der Wert dieser Adresse mit dem "Aktivierungswert" überein, ist das Steuerelement aktiv, so dass es betätigt werden kann.
  • Aktivierungswert: Wert, von dem das Steuerelement aktiviert wird
HINWEIS: Die Aktivierung des Steuerelements kann nur über das benötigte Recht konfiguriert werden oder nur über die Aktivierungsadresse und den Aktivierungswert oder über beide. Werden beide Einstellungen vorgenommen, muss der Benutzer das benötigte Recht besitzen und der Wert der "Aktivierungsadresse" muss mit dem "Aktivierungswert" identisch sein, andernfalls wird das Steuerelement nicht aktiviert.
