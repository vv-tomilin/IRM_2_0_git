Beschreibung

Der Schieberegler dient der grafischen Darstellung des Werts einer Adresse; er kann nicht als Eingabeelement verwendet werden.

Für jeden Schieberegler können 0 bis 4 Limits (von "Unteres Limit unterschritten" bis "Oberes Limit überschritten") festgelegt werden. Diese Limits unterteilen den durch "min." und "max." definierten Bereich in 1 bis 5 Intervalle. Je nachdem, in welchem Intervall der Wert der "base"-Adresse liegt, wird die Textfarbe des Werts gemäß der für jedes Limit definierten Farbe geändert - siehe unten unter "Darstellung". Liegt ein Limit unter "min." oder über "max.", wird es nicht explizit gezeichnet, obwohl die Textfarbe entsprechend geändert wird. (Beispiel: "max." = 100 und "Oberes Limit überschritten" = 120: Obwohl der Schieberegler nicht über "max." hinausgeht, wird die Textfarbe gemäß "Farbe für Oberes Limit überschritten" geändert, sobald der Wert "Oberes Limit überschritten" erreicht.)

Die Limits sind wie folgt definiert:
  • Unteres Limit unterschritten >= Wert: Textfarbe gemäß "Farbe für Unteres Limit unterschritten"
  • Unteres Limit unterschritten < Wert <= Unteres Limit erreicht: Textfarbe gemäß "Farbe für Unteres Limit erreicht"
  • Unteres Limit erreicht < Wert < Oberes Limit erreicht: Textfarbe gemäß "Farbe"
  • Oberes Limit erreicht <= Wert < Oberes Limit überschritten: Textfarbe gemäß "Farbe für Oberes Limit erreicht"
  • Oberes Limit überschritten <= Wert: Textfarbe gemäß "Farbe für Oberes Limit überschritten"

Mit Hilfe einer Statusanzeige in Form eines zusätzlichen Rahmens kann zudem auch ein negativer Status des aktuellen Wertes und ein zusätzlich definierter Alarm dargestellt werden. Die Farben und Blinkintervalle werden in globalen Listen definiert.

Parameter

Der Schieberegler hat die folgenden allgemeinen Parameter:

  • base: Adresse des Datenpunkts, dessen Wert für die grafische Darstellung verwendet wird
  • Alarmadresse: Optional zusätzliche Adresse eines Alarms, welcher angezeigt werden soll.
  • Beschriftung: Beschriftung des Schiebereglers
  • Einheit: Einheit des Werts
  • Minimum: untere Grenze des Werts (Standard: 0)
  • Maximum: obere Grenze des Werts (Standard: 100)
  • Statusüberwachung aktivieren: Schaltet die Anzeige des aktuellen Wertestatus ein oder aus.
  • Statustrigger feuern: Statt der Statusanzeige kann auch ein hier definierter Trigger global gefeuert werden.

Schrift:

  • Schriftart: Schriftart, die auf Wert, Einheit und Beschriftung angewendet wird
  • Textfarbe: Farbe, die auf Wert, Einheit und Beschriftung angewendet wird (Standard: Globaler Parameter atvFontColor)
  • Größe des Werts: auf den Wert angewendete Schriftgröße
  • Größe der Beschriftung: auf die Beschriftung angewendete Schriftgröße
  • Größe der Einheit: auf die Einheit angewendete Schriftgröße

Darstellung:

  • Nachkommastellen: Anzahl der Nachkommastellen für den anzuzeigenden Wert
  • Füllfarbe: Füllfarbe von Hintergrundflächen (Standard: Globaler Parameter atvFillColor)
  • Balkenfarbe: Farbe der Leiste, wenn keine Limits aktiviert wurden (Standard: Globaler Parameter atvAccent1)
  • Balkenfarbe inaktiv: Farbe der Leiste, wenn das Element inaktiv ist
  • Balkenfarbe wenn nicht editierbar: Farbe der Leiste, wenn das Element auf nicht editierbar gesetzt ist
  • Balkenfarbe aktive Limits: Farbe der Leiste, wenn Limit gesetzt und unteres Limit nicht unterschritten und oberes Limit nicht überschritten wurde (Standard: grün)
  • Unteres Limit unterschritten (off=kein Limit): Wert für Unteres Limit unterschritten (Standard: off, d. h. kein Limit)
  • Unteres Limit erreicht (off=kein Limit): Wert für Unteres Limit erreicht (Standard: off, d. h. kein Limit)
  • Oberes Limit erreicht (off=kein Limit):Wert für Oberes Limit erreicht (Standard: off, d. h. kein Limit)
  • Oberes Limit überschritten (off=kein Limit): Wert für Oberes Limit überschritten (Standard: off, d. h. kein Limit)
  • Farbe für Unteres Limit unterschritten: Farbe des Textes, wenn "Unteres Limit unterschritten" >= Wert (Standard: rot)
  • Farbe für Unteres Limit erreicht: Farbe des Textes, wenn "Unteres Limit unterschritten" < Wert <= "Unteres Limit erreicht" (Standard: gelb)
  • Farbe für Oberes Limit erreicht: Farbe des Textes, wenn "Oberes Limit erreicht" <= Wert < "Oberes Limit überschritten" (Standard: gelb)
  • Farbe für Oberes Limit überschritten: Farbe des Textes, wenn "Oberes Limit überschritten" <= Wert (Standard: rot)
  • angezeigte Skala: zu verwendende Skala; weitere Skalen können ausgewählt werden aus "Object Displays (ATVISE)/Symbols/scales (Standard: scale_20_slider)
  • Skalenfarbe: Farbe der zu verwendenden Skala
  • Skalenfarbe klein: Farbe der zu verwendenden kleinen Skala
  • Symbolfarbe: Farbe des Symbols des Schiebereglers (Standard: Globaler Parameter atvSymbolColor)
  • Rahmenfarbe: Farbe des Rahmens um das Steuerelement (Standard: Globaler Parameter atvBorderColor)
  • Statushinweis: Spezifiziert die Darstellung des Status über eine definierte globale Liste (Standard: Globaler Parameter atvStatusIndication)
  • Alarmhinweis: Spezifiziert die Darstellung des Alarms überber eine definierte globale Liste (Standard: Globaler Parameter atvAlarmIndication)
Optionen:
  • Adresse für Minimum: Adresse, der das Minimum entnommen wird; setzt den Parameter "Minimum" außer Kraft
  • Adresse für Maximum: Adresse, der das Maximum entnommen wird; setzt den Parameter "Maximum" außer Kraft
  • Adresse für Unteres Limit unterschritten: Adresse, der "Unteres Limit unterschritten" entnommen wird; setzt den Parameter "Unteres Limit unterschritten" außer Kraft
  • Adresse für Unteres Limit erreicht: Adresse, der "Unteres Limit erreicht" entnommen wird; setzt den Parameter "Unteres Limit erreicht" außer Kraft
  • Adresse für Oberes Limit erreicht: Adresse, der "Oberes Limit erreicht" entnommen wird; setzt den Parameter "Oberes Limit erreicht" außer Kraft
  • Adresse für Oberes Limit überschritten: Adresse, der "Oberes Limit überschritten" entnommen wird; setzt den Parameter "Oberes Limit überschritten" außer Kraft
