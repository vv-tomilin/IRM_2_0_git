Description

This display contains a button for saving, browsing and deleting bookmarks.

Parameters

This control has the following generic parameters:

  • configuration node: node address for saving favorites data
  • size limit: Saved char limit for the configuration node
Context menu:
  • font family: font family which will be applied to label (default: Arial)
  • font size: font size which will be applied to the label (default: 16)
  • font color: color which will be applied to the label (default: global parameter atvFontColor)
  • background color: background color of context menu's option elements
  • hover color: background color of context menu's option elements at mouseover.
  • text alignment: alignment of the context menu's texts
Popups:
  • font family: font family which will be applied to text in popups (default: Arial)
  • font size: font size which will be applied to text in popups (default: 14)
  • font color: color which will be applied to text in popups (default: global parameter atvFontColor)
  • background color: background color in popups (default: global parameter atvFillColor2)
Button:
  • symbol color: color of the symbol to be used (default: global parameter atvSymbolColor)
  • symbol scaling (%): factor for resizing the symbol (default: 100)
  • necessary right: the user must have this right to activate this button, i.e. to press the button
  • activation address: if the value of this address equals to "activation value", then the button will be active, i.e. can be pressed
  • activation value: value which activates the button
