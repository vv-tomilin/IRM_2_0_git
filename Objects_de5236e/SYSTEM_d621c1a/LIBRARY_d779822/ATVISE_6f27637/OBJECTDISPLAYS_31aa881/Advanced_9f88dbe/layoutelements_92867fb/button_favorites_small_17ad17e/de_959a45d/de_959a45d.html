Beschreibung

Dieses Display beinhaltet eine Schaltfläche zum Speichern, Browsen und Löschen von Lesezeichen.

Parameter

Dieses Steuerelement hat die folgenden allgemeinen Parameter:

  • Konfigurationsnode: Adresse des Nodes für die Speicherung der Favoriten-Daten
  • Größenlimitierung: Limitierung der maximal abgespeicherten Zeichen in der Node
Kontextmenü:
  • Schriftart: Schriftart, die auf die Beschriftung angewendet wird (Standard: Arial)
  • Schriftgröße: Schriftgröße, die auf die Beschriftung der Schaltfläche angewendet wird (Standard: 16)
  • Schriftfarbe: Farbe, die auf die Beschriftung angewendet wird (Standard: Globaler Parameter atvFontColor)
  • Hintergrundfarbe: Hintergrundfarbe der Optionselemente des Kontextmenüs
  • Farbe Hervorhebung: Hintergrundfarbe eines Optionselements des Kontextmenüs, bei MouseOver.
  • Textausrichtung: Ausrichtung des Kontextmenü-Texts
Popup:
  • Schriftart: Schriftart, die auf Schrifen in Popups angewendet wird (Standard: Arial)
  • Schriftgröße: Schriftgröße, die auf Schrifen in Popups angewendet wird (Standard: 14)
  • Schriftfarbe: Farbe, die auf Schriften in Popups angewendet wird (Standard: Globaler Parameter atvFontColor)
  • Hintergrundfarbe: Hintergrundfarbe in Popups (Standard: Globaler Parameter atvFillColor2)
Schaltfläche Allgemein:
  • Symbolfarbe: Farbe des zu verwendenden Symbols (Standard: Globaler Parameter atvSymbolColor)
  • Symbolskalierung: Skalierung des Symbols in Prozent (Standard: 100)
  • Benötigtes Recht: Der Benutzer muss über dieses Recht verfügen, um diese Schaltfläche aktivieren, d. h. betätigen zu können.
  • Aktivierungsadresse: Stimmt der Wert dieser Adresse mit dem "Aktivierungswert" überein, ist die Schaltfläche aktiv, so dass sie betätigt werden kann.
  • Aktivierungswert: Wert, von dem die Schaltfläche aktiviert wird
