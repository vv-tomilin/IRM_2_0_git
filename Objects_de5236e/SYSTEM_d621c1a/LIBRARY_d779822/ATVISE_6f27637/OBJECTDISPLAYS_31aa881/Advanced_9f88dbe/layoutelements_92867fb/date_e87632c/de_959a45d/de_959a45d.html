Beschreibung

Dieses Display zeigt das aktuelle Datum.

Parameter

Dieses Steuerelement hat die folgenden allgemeinen Parameter:

  • Datumsformat: Das Datum kann mithilfe von Platzhaltern formatiert werden (Standard: %d.%m.%Y)

    • %d: Platzhalter für den Tag im 2-Ziffern-Format
    • %m: Platzhalter für das Monat im 2-Ziffern-Format
    • %y: Platzhalter für das Jahr im 2-Ziffern-Format
    • %Y: Platzhalter für das Jahr im 4-Ziffern-Format

Schrift:
  • Schriftart: Schriftart, die auf den Text angewendet wird (Standard: Arial)
  • Schriftgröße: Schriftgröße, die auf den Text angewendet wird (Standard: 18)
  • Schriftfarbe: Farbe, die den Text angewendet wird (Standard: Globaler Parameter atvFontColor)
