Description

This display shows the current language and contains a button for changing the language.

Parameters

This control has the following generic parameters:

  • background color: background color of language box (default: global parameter atvFillColor2)
Font:
  • font family: font family which will be applied to text and the button's label (default: Arial)
  • font size: font size which will be applied to text and the button's label (default: 18)
  • font color: color which will be applied to text and the button's label (default: global parameter atvFontColor)
Context menu:
  • font family: font family which will be applied to texts of the context menu (default: Arial)
  • font size: font size which will be applied to texts of the context menu (default: 12)
  • font color: color which will be applied to texts of the context menu (default: global parameter atvFontColor)
  • background color: background color of context menu's option elements (default: global parameter atvFillColor)
  • hover color: background color of context menu's option elements at mouseover (default: global parameter atvFillColor2)
  • text alignment: alignment of the context menu's texts (default: center)
