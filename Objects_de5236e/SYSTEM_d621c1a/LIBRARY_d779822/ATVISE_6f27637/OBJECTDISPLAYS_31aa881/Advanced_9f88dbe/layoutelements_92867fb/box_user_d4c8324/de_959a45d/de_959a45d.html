Beschreibung

Dieses Display zeigt die aktuelle Sprache an und beinhaltet eine Schaltfläche zur Änderung der Sprache.

Parameter

Dieses Steuerelement hat die folgenden allgemeinen Parameter:

  • Hintergrundfarbe: Hintergrundfarbe der Box für Sprache (Standard: Globaler Parameter atvFillColor2)
Schrift:
  • Schriftart: Schriftart, die auf Text und die Beschriftung der Schaltfläche angewendet wird (Standard: Arial)
  • Schriftgröße: Schriftgröße, die auf Text und die Beschriftung der Schaltfläche angewendet wird (Standard: 18)
  • Schriftfarbe: Farbe, die auf Text und die Beschriftung der Schaltfläche angewendet wird (Standard: Globaler Parameter atvFontColor)
