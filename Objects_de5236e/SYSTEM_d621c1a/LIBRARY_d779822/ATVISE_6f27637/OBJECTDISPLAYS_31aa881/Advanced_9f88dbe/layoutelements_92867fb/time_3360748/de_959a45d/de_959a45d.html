Beschreibung

Dieses Display zeigt die aktuelle Zeit.

Parameter

Dieses Steuerelement hat die folgenden allgemeinen Parameter:

  • Zeitformat: Die Zeit kann mithilfe von Platzhaltern formatiert werden (Standard: %H:%M)

    • %H: Platzhalter für die Stunde im 2-Ziffern-Format
    • %M: Platzhalter für die Minuten im 2-Ziffern-Format
    • %S: Platzhalter für die Sekunden im 2-Ziffern-Format

Schrift:
  • Schriftart: Schriftart, die auf den Text angewendet wird (Standard: Arial)
  • Schriftgröße: Schriftgröße, die auf den Text angewendet wird (Standard: 38)
  • Schriftfarbe: Farbe, die den Text angewendet wird (Standard: Globaler Parameter atvFontColor)
