Beschreibung

Dieses Display ermöglicht die Eingabe von Favoriten-Namen.

Parameter

Schrift:

  • Schriftart: Schriftart, die auf den Text angewendet wird (Standard: Arial)
  • Schriftgröße: Schriftgröße, die auf den Text angewendet wird (Standard: 14)
  • Schriftfarbe: Farbe, die den Text angewendet wird (Standard: Globaler Parameter atvFontColor)
Allgemeiner Parameter:
  • Hintergrundfarbe: Die Füllfarbe des Formular-Hintergrunds (Standard: Globaler Parameter atvFillColor2)
