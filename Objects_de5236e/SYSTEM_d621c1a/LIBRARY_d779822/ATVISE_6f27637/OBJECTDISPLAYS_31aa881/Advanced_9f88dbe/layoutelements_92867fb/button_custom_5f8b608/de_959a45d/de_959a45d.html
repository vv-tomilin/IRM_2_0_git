Beschreibung

Dieses Display beinhaltet eine konfigurierbare Schaltfläche zur Navigation.

Parameter

Schaltfläche 1:

  • Beschriftung Zeile 1: Text der ersten Zeile der Schaltfläche
  • Beschriftung Zeile 2: Text der zweiten Zeile der Schaltfläche. Ist dieser leer, so wird Beschriftung Zeile 1 horizontal zentriert angezeigt.
  • Symbol: Zu verwendendes Symbol; weitere Symbole können ausgewählt werden aus "Object Displays (ATVISE)/Symbols/20x20 (Standard: prüfen)
  • Symbolskalierung (%): Faktor für die Vergößerung oder Verkleinerung des Symbols (Standard: 200)
  • Zieldisplay: Display, das bei Klick auf die Schaltfläche aufgerufen wird
  • Kontextmenü: Globale Liste, die die Daten des Kontextmenüs enthält, das bei Klick auf die Schaltfläche geöffnet wird. Ist ein Kontextmenü angegeben, so bleibt das Zieldisplay wirkungslos.
  • Trigger: Triggerfunktion, die bei Klick auf die Schaltfläche aufgerufen wird
  • Schriftgröße: Schriftgröße, die auf die Beschriftung der Schaltfläche angewendet wird (Standard: 18)
  • Schriftfarbe: Farbe, die auf die Beschriftung angewendet wird (Standard: Globaler Parameter atvFontColor)
  • Benötigtes Recht: Der Benutzer muss über dieses Recht verfügen, um diese Schaltfläche aktivieren, d. h. betätigen zu können.
  • Aktivierungsadresse: Stimmt der Wert dieser Adresse mit dem "Aktivierungswert" überein, ist die Schaltfläche aktiv, so dass sie betätigt werden kann.
  • Aktivierungswert: Wert, von dem die Schaltfläche aktiviert wird
Schaltfläche 2:
  • Beschriftung Zeile 1: Text der ersten Zeile der Schaltfläche
  • Beschriftung Zeile 2: Text der zweiten Zeile der Schaltfläche. Ist dieser leer, so wird Beschriftung Zeile 1 horizontal zentriert angezeigt.
  • Symbol: Zu verwendendes Symbol; weitere Symbole können ausgewählt werden aus "Object Displays (ATVISE)/Symbols/20x20 (Standard: prüfen)

Kontextmenü:
  • Schriftart: Schriftart, die auf die Texte des Kontextmenüs angewendet wird (Standard: Arial)
  • Schriftgröße: Schriftgröße, die auf die Texte des Kontextmenüs angewendet wird (Standard: 18)
  • Schriftfarbe: Farbe, die auf die Texte des Kontextmenüs angewendet wird (Standard: Globaler Parameter atvFontColor)
  • Hintergrundfarbe: Hintergrundfarbe der Optionselemente des Kontextmenüs (Standard: Globaler Parameter atvFillColor)
  • Farbe Hervorhebung: Hintergrundfarbe eines Optionselements des Kontextmenüs, bei MouseOver (Standard: Globaler Parameter atvFillColor2)
  • Textausrichtung: Ausrichtung des Kontextmenü-Texts (Standard: center)
