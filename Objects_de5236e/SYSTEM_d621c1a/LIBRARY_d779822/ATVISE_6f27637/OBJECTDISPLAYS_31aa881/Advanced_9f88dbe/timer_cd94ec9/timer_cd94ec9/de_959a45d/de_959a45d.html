Zeitschaltuhr

Eine ausführliche Hilfe zur Bedienung der Zeitschaltuhr finden Sie in der atvise Dokumentation.

Parameter

Es stehen folgende Parameter zur Verfügung:

  • Textfarbe: Farbe, die auf Beschriftungen angewendet wird (Standard: Globaler Parameter atvFontColor)
  • Füllfarbe: Füllfarbe von Hintergrundflächen (Standard: Globaler Parameter atvFillColor)
