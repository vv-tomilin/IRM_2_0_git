Beschreibung

Dieses Steuerelement ermöglicht die Eingabe eines Datums.

Darüber hinaus wird der Trigger "valuechanged" aktiviert.

Parameter

Dieses Steuerelement hat die folgenden allgemeinen Parameter:

  • base: Der Wert dieser Adresse wird auf das ausgewählte Datum gesetzt.
  • Alarmadresse: Optional zusätzliche Adresse eines Alarms, welcher angezeigt werden soll.
  • Format: Die Reihenfolge der Eingabefelder ist entweder "dd.mm.yyyy" oder "mm.dd.yyyy".
  • Intervall [ms] (0 = aus): Legt fest, wie lange das Steuerelement betätigt werden muss, damit der Wert von "base" automatisch geändert wird (Standard: 150 ms, d. h., wird die Schaltfläche kontinuierlich betätigt, wird der Wert alle 150 ms geändert).
  • Statusüberwachung aktivieren: Schaltet die Anzeige des aktuellen Wertestatus ein oder aus.
  • Statustrigger feuern: Statt der Statusanzeige kann auch ein hier definierter Trigger global gefeuert werden.
Schrift:
  • Schriftart: Schriftart, die auf den Wert angewendet wird (Standard: Arial)
  • Größe: Schriftgröße, die auf den Wert angewendet wird (Standard: 12)
  • Farbe: Farbe, die auf den Wert angewendet wird (Standard: Globaler Parameter atvFontColor)
Darstellung:
  • Füllfarbe Wertefeld: Füllfarbe der drei Wertefelder (Standard: weiß)
  • Füllfarbe Wertefeld inaktiv: Füllfarbe der Wertefelder, wenn nicht aktiv (Standard: dunkelgrau)
  • Füllfarbe Schaltfläche: Füllfarbe der sechs Schaltflächen (Standard: schwarz)
  • Füllfarbe Pfeil: Füllfarbe des Pfeils (Standard: grau)
  • Füllfarbe gedrückter Pfeil: Füllfarbe der Pfeile, wenn sie betätigt werden (Standard: gelb)
  • Konturfarbe Schaltfläche gedrückt: Farbe der Kontur, die bei Betätigung einer der sechs Schaltflächen eingestellt wird (Standard: weiß)
  • Fokus Konturfarbe: Farbe der Kontur, die beim Fokus der Schaltfläche eingestellt wird (Standard: weiß)
Optionen:
  • Minimum: Legt das untere Limit für das zu ändernde Datum fest (z. B. 1.1.2011), d. h., wenn das aktuelle Datum mit diesem Minimum übereinstimmt, wird es nicht weiter zurückgesetzt (Standard: kein Minimum festgelegt).
  • Maximum: Legt das obere Limit für das zu ändernde Datum fest (z. B. 31.12.2011), d. h., wenn das aktuelle Datum mit diesem Maximum übereinstimmt, wird es nicht weiter vorgesetzt (Standard: kein Maximum festgelegt).
  • Zu setzende Adresse: Mit diesem Parameter kann, parallel zu "base" eine zweite Adresse eingestellt werden.
  • Konsistenzgruppe: Name der Konsistenzgruppe.
  • Tab-Index: Bestimmt die Reihenfolge, in der mit der Tabulatortaste zwischen den einzelnen Object Displays gewechselt werden kann
  • Tooltip: Zeigt eine Tooltip Box mit dem eingegebenen Text an (Standard HTML-Elemente können auch eingegeben werden)
Sicherheit:
  • Benötigtes Recht: Der Benutzer muss über dieses Recht verfügen, um dieses Steuerelement aktivieren, d. h. eine der sechs Schaltflächen betätigen zu können.
  • Aktivierungsadresse: Stimmt der Wert dieser Adresse mit dem "Aktivierungswert" überein, ist das Steuerelement aktiv, so dass es betätigt werden kann.
  • Aktivierungswert: Wert, von dem das Steuerelement aktiviert wird
HINWEIS: Die Aktivierung des Steuerelements kann nur über das benötigte Recht konfiguriert werden oder nur über die Aktivierungsadresse und den Aktivierungswert oder über beide. Werden beide Einstellungen vorgenommen, muss der Benutzer das benötigte Recht besitzen und der Wert der "Aktivierungsadresse" muss mit dem "Aktivierungswert" identisch sein, andernfalls wird das Steuerelement nicht aktiviert.
