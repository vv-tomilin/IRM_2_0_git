Description

This display is part of the "opt_series" display and provide specific settings for the bar and column diagram.

Options

  • Related x/y-axis: Defines the related x- and y-axes for the actual data set.

Styling

  • Border color: Defines the color of the border line of the bar or column.
  • Fill color: Defines the fill color of the bar or column.
  • Border line width: Defines the width of the border line.
  • Labels: Enables or disables the visibility of the labels.
  • Label format: Format of the label, for more information check the Highcharts API reference.

Parameters

This control has the following parameter:

  • font color: color which will be applied to the labels (default: global parameter atvFontColor)
