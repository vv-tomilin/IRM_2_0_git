Beschreibung

Mit Hilfe dieses Display lassen sich allgemeine Einstellungen für das Diagramm festlegen.

Einstellungen

  • Diagrammtitel: Titel des Diagramms.
  • Messzeiger: Aktiviert oder deaktiviert die Anzeige der Messzeiger.
  • Tooltips: Zeigt Tooltips zu den einzelnen Datenpunkten im Diagramm.
  • Exportmenü: Zeigt oder versteckt das Exportmenü an der oberen rechten Ecke.
  • Aktualisierungsintervall: Zeitraum zwischen dem Hinzufügen von neuen Daten im Diagramm.
  • Datenquelle: Definiert das Datenquellenmodul für das Importieren der Daten.
  • Datenmodus: Als Modus kann "Gesamt", "nur Live" oder "nur Archive" ausgewählt werden.
  • Zeitformat: Definiert das Zeitformat der angezeigten Daten im Diagramm.
  • Downsampling: Downsampling wird verwendet um die Leistung, gerade für schwächere Geräte, zu optimieren. Dabei werden die angezeigten Datenpunkte gemäß Bildschirmauflösung und Diagrammgröße reduziert.

Parameter

Es stehen folgende Parameter zur Verfügung:

  • Schriftfarbe: Farbe, die auf die Beschriftung angewendet wird (Standard: Globaler Parameter atvFontColor)
  • Füllfarbe: Füllfarbe der linken Seitenleiste (Standard: Globaler Parameter atvFillColor)
