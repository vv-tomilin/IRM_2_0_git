Beschreibung

Dies ist das Hauptdisplay des Konfigurationsdialogs. Der Konfigurationsdialog kann mit Hilfe der Toolbar geönnet werden. Das Display besteht im wesentlichen aus einer Navigationsleiste um das ausgewählte Diagramm entsprechend konfigurieren zu können.

Parameter

Die folgenden Parameter sind verfügbar:

  • Diagrammname: Eindeutiger Name zur Identifizierung der Diagramminstanz.

Parameter Darstellung

  • Füllfarbe: Füllfarbe von Hintergrundflächen (Standard: Globaler Parameter atvFillColor)
  • Schriftfarbe: Schriftfarbe (Standard: Globaler Parameter atvFontColor)
  • Zeige "Vorschau" Schaltfläche (nur bei Popup): Aktiviert die Schaltfläche "Vorschau", wenn der Konfigurationsdialog in einem Popup geöffnet wurde.
  • Zeige "Zurück" Schaltfläche: Aktiviert die Schaltfläche "Zurück", wenn der Konfigurationsdialog in einem Frame geöffnet wurde.

Parameter Optionen

  • URL für "Zurück": Gibt das Display an, welches geöffnet wird, wenn auf die Schaltfläche "Zurück" gedrückt wird.
  • Frame für "Zurück" Schaltfläche: Gibt den Framenamen an, in dem das ausgewählte Display geöffnet werden soll.

Parameter Sicherheit

Außerdem können über ensprechende Parameter die benötigten Rechte zum Laden und Speichern der Konfiguration vergeben werden:

  • Benötigtes Recht: Der Benutzer muss über dieses Recht verfügen, um die Funktion ausführen zu können.
  • Aktivierungsadresse: Stimmt der Wert dieser Adresse mit dem "Aktivierungswert" überein, ist die Funktion aktiviert.
  • Aktivierungswert: Wert, von dem die Funktion aktiviert wird.
