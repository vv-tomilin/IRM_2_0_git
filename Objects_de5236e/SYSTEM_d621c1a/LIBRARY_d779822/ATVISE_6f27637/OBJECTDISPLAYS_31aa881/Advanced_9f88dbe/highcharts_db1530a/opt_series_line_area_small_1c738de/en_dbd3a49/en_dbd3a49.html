Description

This display is part of the "opt_series" display and provide specific settings for the diagram types line and area.

Options

  • Related x/y-axis: Defines the related x- and y-axes for the actual data set.
  • Interpolation: Specifies the interpolation method: linear, spline and step.
  • Missing data: Defines if missing data points are ignored and shown as gap or connected lines.
  • Non-stop: The line will be extended beyond the first and the last visible data points.
  • Marker: Enables the display of the data points.

Styling

  • Line color: Defines the color of the line.
  • Fill color: Defines the color of the area between the two lines within an area diagram.
  • Line width: Defines the width of the line.
  • Line type: Defines the type of the line: solid, dotted or dashed.

Parameters

This control has the following parameter:

  • font color: color which will be applied to the labels (default: global parameter atvFontColor)
