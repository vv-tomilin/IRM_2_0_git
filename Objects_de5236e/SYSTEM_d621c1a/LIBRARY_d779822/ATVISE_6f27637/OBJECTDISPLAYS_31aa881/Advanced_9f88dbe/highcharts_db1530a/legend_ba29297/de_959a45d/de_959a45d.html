Beschreibung

Diese Legende zeigt Informationen über die im Diagramm angezeigten Datensätze in Form einer Tabelle an:

  • Name
  • Adresse
  • Aggregat
  • Typ
  • Wert
  • Sichtbarkeit
  • Farbe
  • Marker 1 X-Wert
  • Marker 2 X-Wert
  • Delta X-Wert
  • Marker 1 Y-Wert
  • Marker 2 Y-Wert
  • Delta Y-Wert
  • Einstellungen

Standardmäß sind einige Spalten ausgeblendet, diese können mit den unten dargestellten Parametern aktiviert werden.

Zusätzlich können die Liniensichtbarkeit und die Linienfarbe im Diagramm direkt über die Tabelle geändert werden.

Parameter

Für die Legende stehen die folgenden Parameter zur Verfügung:

  • Diagrammname: Eindeutiger Name der Diagramminstanz.
  • Benötigtes Recht:
  • Format Datum-Zeit: Format zur Darstellung des gemessenen X-Wertes.
  • Format Zeitdifferenz: Format zur Darstellung des Zeitdifferenz zwischen den beiden Messwerten.

Parameter für Spalten

Für die oben angeführten Spalten gibt es außerdem jeweils die Möglichkeit folgende einzelne Parameter zu konfigurieren:

  • den Namen: Benutzerdefinierter Spaltenname
  • die Sichtbarkeit: Sichtbarkeit der Spalte
  • den Index: Bestimmung der Reihenfolge der Spalten
  • die Breite: Fixierung der Spaltenbreite

Parameter Darstellung

  • Tabellenkopf Hintergrundfarbe: Hintergrundfarbe für den Tabellenkopf (Hex-Wert).
  • Tabellenfuß Hintergrundfarbe: Hintergrundfarbe für den Tabellenfuß (Hex-Wert).
  • Hervorhebung Hintergrundfarbe: Farbe für Hervorhebungen (Hex-Wert).
  • Hervorhebung Auswahlfarbe: Farbe für Auswahl (Hex-Wert).

Parameter Schrift

  • Tabellenkopf Schriftfarbe: Schriftfarbe im Tabellenkopfs.
  • Tabellenfuß Schriftfarbe: Schriftfarbe im Tabellenfuß.
