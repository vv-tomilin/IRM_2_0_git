Beschreibung

Über die Combobox können Sie Elemente aus einer Liste auswählen.

Je nach Einstellung des Parameters "Modus" kann sie in den nachstehenden Modi verwendet werden:

  1. Text: Sowie dieser Modus ausgewählt wird, müssen die Elemente in der Gruppe "Textmodus" konfiguriert werden. Jedes Element der Combobox besteht aus einem "Textelement" und einem "Wertelement". Das "Textelement" wird als Element in der Combobox angezeigt, während das "Wertelement" in "base" oder "Alternative Ausgabeadresse" geschrieben wird, wenn das zugehörige "Textelement" ausgewählt wird. In diesem Modus können bis zu fünf Elemente konfiguriert werden. Wenn Sie mehr als fünf Elemente konfigurieren wollen, müssen Sie den Modus "Enum Liste" oder "Script" verwenden.
  1. Boolean: In der Combobox werden die beiden Elemente "true" und "false" angezeigt. Die "base" wird entsprechend dem ausgewählten Element gesetzt und muss daher vom Typ "Boolean" sein.
  1. Enum Liste: Sowie dieser Modus ausgewählt wird, müssen die Elemente in der Gruppe "Enum List Modus" konfiguriert werden. Jedes Element in der Combobox besteht gemäß der definierten enum Liste aus einem "Text" und einem "Wert". Der "Text" wird als Element in der Combobox angezeigt, während der "Wert" in "base" oder "Alternative Ausgabeadresse" geschrieben wird, wenn der zugehörige "Text" ausgewählt wird.
  1. Script: Dieser Modus ermöglicht die Behandlung der Elemente der Combobox mit den folgenden Triggern in einem JavaScript:
  • setItems: Dient dem Definieren (Initialisieren) der Combobox mit dem vordefinierten Objekt, das alle Elemente (Paare aus Text und Wert) enthält.
  • setSelectedItem: Dient dem Definieren eines Elements (nur Text) in der Combobox als ausgewählt. 
  • addItem: Die in dem Hinzufügen eines Elements (Paar aus Text und Wert) zur Combobox.
  • getSelectedItem: Aktiviert den Trigger "selectedItem", wenn der Parameter vom Typ String ist, oder ruft die callback function mit dem Wert des derzeit in der Combobox ausgewählten Elements auf.
  • getItems: Aktiviert den Trigger "items" mit einem Array, das alle Elemente (Paare von Text und Wert) der Combobox enthält.
Ein Beispiel finden Sie im nachstehenden Abschnitt "Beispiele".

Mit Hilfe einer Statusanzeige in Form eines zusätzlichen Rahmens kann zudem auch ein negativer Status des aktuellen Wertes und ein zusätzlich definierter Alarm dargestellt werden. Die Farben und Blinkintervalle werden in globalen Listen definiert.

Parameter

Dieses Steuerelement hat die folgenden allgemeinen Parameter:

  • base: Je nachdem, welcher der oben beschriebenen Modi ausgewählt ist, wird die "base" auf den entsprechenden Wert des ausgewählten Elements in der Combobox gesetzt.
  • Alarmadresse: Optional zusätzliche Adresse eines Alarms, welcher angezeigt werden soll.
  • Modus: Wählen Sie einen der oben beschriebenen vier Modi aus (Standard: Text).
  • alternative Ausgabeadresse: Sowie diese Adresse konfiguriert ist, wird der Wert in sie und nicht in die "base" geschrieben.
  • Statusüberwachung aktivieren: Schaltet die Anzeige des aktuellen Wertestatus ein oder aus.
  • Statustrigger feuern: Statt der Statusanzeige kann auch ein hier definierter Trigger global gefeuert werden.
Textmodus:
  • Text Element 1: erstes in der Combobox anzuzeigendes Textelement
  • Wert Element 1: mit dem ersten Textelement verknüpfter Wert
  • Text Element 2: zweites in der Combobox anzuzeigendes Textelement
  • Wert Element 2: mit dem zweiten Textelement verknüpfter Wert
  • Text Element 3: drittes in der Combobox anzuzeigendes Textelement
  • Wert Element 3: mit dem dritten Textelement verknüpfter Wert
  • Text Element 4: viertes in der Combobox anzuzeigendes Textelement
  • Wert Element 4: mit dem vierten Textelement verknüpfter Wert
  • Text Element 5: fünftes in der Combobox anzuzeigendes Textelement
  • Wert Element 5: mit dem fünften Textelement verknüpfter Wert
Enum List Modus:
  • enum Liste: Globale Liste, die die Paare (Text + Wert) für die Combobox definiert, siehe Modus 3 oben.
Schrift:
  • Schriftart: Schriftart, die auf den Wert angewendet wird (Standard: Arial)
  • Größe: Schriftgröße, die auf den Wert angewendet wird (Standard: 12)
  • Farbe:Farbe, die auf den Wert angewendet wird (Standard: Globaler Parameter atvFontColor)
Darstellung:
  • Füllfarbe: Füllfarbe von Hintergrundflächen (Standard: Globaler Parameter atvFillColor)
  • Füllfarbe Wertefeld: Füllfarbe des Wertefelds (Standard: weiß)
  • Füllfarbe Wertefeld inaktiv: Füllfarbe des Wertefelds, wenn nicht aktiv (Standard: dunkelgrau)
  • Symbolfarbe: Farbe des Symbols (Standard: Globaler Parameter atvSymbolColor)
  • Konturfarbe gedrückt: Farbe der Kontur, die beim Klicken in die Combobox eingestellt wird (Standard: weiß)
  • Fokus Konturfarbe: Farbe der Kontur, die beim Fokus der Schaltfläche eingestellt wird (Standard: weiß)
  • Rahmenfarbe: Farbe des Rahmens um das Steuerelement (Standard: Globaler Parameter atvBorderColor)
  • Statushinweis: Spezifiziert die Darstellung des Status über eine definierte globale Liste (Standard: Globaler Parameter atvStatusIndication)
  • Alarmhinweis: Spezifiziert die Darstellung des Alarms überber eine definierte globale Liste (Standard: Globaler Parameter atvAlarmIndication)
Sicherheit:
  • Benötigtes Recht: Der Benutzer muss über dieses Recht verfügen, um diese Combobox aktivieren zu können.
  • Aktivierungsadresse: Stimmt der Wert dieser Adresse mit dem "Aktivierungswert" überein, ist die Combobox aktiv, so dass auf sie geklickt werden kann.
  • Aktivierungswert: Wert, von dem die Combobox aktiviert wird
HINWEIS: Die Aktivierung der Combobox kann nur über das benötigte Recht konfiguriert werden oder nur über die Aktivierungsadresse und den Aktivierungswert oder über beide. Werden beide Einstellungen vorgenommen, muss der Benutzer das benötigte Recht besitzen und der Wert der "Aktivierungsadresse" muss mit dem "Aktivierungswert" identisch sein, andernfalls wird die Combobox nicht aktiviert.

Optionen:
  • Maximal angezeigte Elemente (0 = alle): Höchstzahl der Elemente, die beim Blättern einer Seite angezeigt werden (0 = Blättern aus, Standard: 0)
  • Konsistenzgruppe: Name der Konsistenzgruppe
  • Tab-Index: Bestimmt die Reihenfolge, in der mit der Tabulatortaste zwischen den einzelnen Object Displays gewechselt werden kann
  • Tooltip: Zeigt eine Tooltip Box mit dem eingegebenen Text an (Standard HTML-Elemente können auch eingegeben werden)

Beispiele

Die folgenden Code-Elemente zeigen, wie die Combobox mit den Triggern bearbeitet wird:

setItems
var entries = [
  { text: "Entry 1", value: "1" },
  { text: "Entry 2", value: "2" },
  { text: "Entry 3", value: "3" }
];
webMI.trigger.fire("setItems", entries, "myCombobox");

setSelectedItem
webMI.trigger.fire("setSelectedItem", "Entry 1", "myCombobox");

addItem
webMI.trigger.fire("addItem", { text: "Entry 4", value: "4" }, "myCombobox");

getSelectedItem (aktiviert den Trigger "selectedItem")
webMI.trigger.fire("getSelectedItem", function(e){alert(e.text+" "+e.value);},"myCombobox");

oder

webMI.trigger.connect("selectedItem", function(e) { alert(e.value); }, "myCombobox");
webMI.trigger.fire("getSelectedItem", "", "myCombobox");

getItems (aktiviert den Trigger "items")
webMI.trigger.connect("items", function(e) { alert(e.value); }, "myCombobox");
webMI.trigger.fire("getItems", "", "myCombobox");
