Beschreibung

Diese Schaltfläche führt Aktionen auf die Elemente in der Alarmliste aus.

Parameter

Dieses Steuerelement hat die folgenden allgemeinen Parameter:

  • Tabellen-ID: ID der entsprechenden Alarmliste
  • Aktion: Definiert die durchzuführende Aktion auf die Elemente der Alarmliste.
  • Beschriftung: Beschriftung der Schaltfläche, wenn "Sichtbarkeit Beschriftung" auf "visible" eingestellt ist
Schrift:
  • Schriftart: Schriftart, die auf die Beschriftung angewendet wird (Standard: Arial)
  • Größe: Schriftgröße, die auf die Beschriftung angewendet wird (Standard: 12)
  • Farbe: Farbe, die auf die Beschriftung angewendet wird (Standard: Globaler Parameter atvFontColor)
Darstellung:
  • Sichtbarkeit Beschriftung: Bei Einstellung auf "visible" wird der über "Beschriftung" definierte Text angezeigt; bei Einstellung auf "visible" sollte "Sichtbarkeit Symbol" auf "hidden" eingestellt sein (Standard: visible).
  • Sichtbarkeit Symbol: Bei Einstellung auf "visible" wird das über "Symbol" definierte Symbol angezeigt; bei Einstellung auf "visible" sollte "Sichtbarkeit Beschriftung" auf "hidden" eingestellt sein (Standard: hidden).
  • Symbol: zu verwendendes Symbol, wenn "Sichtbarkeit Symbol" auf "visible" eingestellt ist; weitere Symbole können ausgewählt werden aus "Object Displays (ATVISE)/Symbols/20x20 (Standard: prüfen)
  • Symbolfarbe: Farbe des zu verwendenden Symbols (Standard: Globaler Parameter atvSymbolColor)
  • Füllfarbe: Füllfarbe der Schaltfläche (Standard: Globaler Parameter atvFillColor)
  • Füllfarbe wenn inaktiv: Füllfarbe der nicht aktiven Schaltfläche (Standard: dunkelgrau)
  • Konturfarbe gedrückt: Farbe der Kontur, die bei Betätigung der Schaltfläche eingestellt wird (Standard: weiß)
  • Fokus Konturfarbe: Farbe der Kontur, die beim Fokus der Schaltfläche eingestellt wird (Standard: weiß)
  • Rahmenfarbe: Farbe des Rahmens um die Schaltfläche (Standard: Globaler Parameter atvBorderColor)
  • Farbe Zusatzrahmen: Farbe des Zusatzrahmens innerhalb des Buttons (Standard: none)
Sicherheit:
  • Benötigtes Recht: Der Benutzer muss über dieses Recht verfügen, um diese Schaltfläche aktivieren, d. h. betätigen zu können.
  • Aktivierungsadresse: Stimmt der Wert dieser Adresse mit dem "Aktivierungswert" überein, ist die Schaltfläche aktiv, so dass sie betätigt werden kann.
  • Aktivierungswert: Wert, von dem die Schaltfläche aktiviert wird
HINWEIS: Die Aktivierung einer Schaltfläche kann nur über das benötigte Recht konfiguriert werden oder nur über die Aktivierungsadresse und den Aktivierungswert oder über beide. Werden beide Einstellungen vorgenommen, muss der Benutzer das benötigte Recht besitzen und der Wert der "Aktivierungsadresse" muss mit dem "Aktivierungswert" identisch sein, andernfalls wird die Schaltfläche nicht aktiviert.

Konsistenz:
  • Gruppe: Name der Konsistenzgruppe
  • Funktion: Funktion, die bei Klick ausgeführt werden soll, sofern eine Gruppe angegeben ist (Standard: write)
Optionen:
  • Tab-Index: Bestimmt die Reihenfolge, in der mit der Tabulatortaste zwischen den einzelnen Object Displays gewechselt werden kann
  • Tooltip: Zeigt eine Tooltip Box mit dem eingegebenen Text an (Standard HTML-Elemente können auch eingegeben werden)
