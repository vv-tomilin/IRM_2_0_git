Beschreibung

Dieses Display ergänzt die Tabelle um einen SVG Keyboard. Eine spezielle Konfiguration, wie für die Tabelle, ist für dieses Display nicht erforderlich. Es muss lediglich der Tabellenname und Eingabeart in den Parametern eingestellt werden.

Anforderungen

Dieses Display kann nur zusammen mit einer Tabelle eingesetzt werden!

Parameter

  • Tabellen-ID: Eindeutiger Tabellenname
  • Schriftfamilie: Rerviert für Schriftfamilie (Standard: Arial)
  • Schriftgröße: Rerviert für Schriftgröße (Standard: 12)
  • Farbe: Rerviert für Schriftfarbe und Iconfarbe (Standard: SYSTEM.GLOBALS.atvSymbolColor)
  • direkte Eingabe erlauben: Auswahl zwischen direkter Eingabe und Eingabe über SVG Tastatur.
