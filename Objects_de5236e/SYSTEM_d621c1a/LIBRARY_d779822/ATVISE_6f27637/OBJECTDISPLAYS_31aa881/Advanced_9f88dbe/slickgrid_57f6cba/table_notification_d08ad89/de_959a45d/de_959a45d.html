Beschreibung

Dieses Display ergänzt die Tabelle um einen Notificaton - Button zum Aufruf von zusätzlichen Informationen im z.B. Fehlerfall. Eine spezielle Konfiguration, wie für die Tabelle, ist für dieses Display nicht erforderlich. Es muss lediglich der Tabellenname und Schriftart in den Parametern eingestellt werden.

Anforderungen

Dieses Display kann nur zusammen mit einer Tabelle eingesetzt werden!

Parameter

  • Tabellen-ID: Eindeutiger Tabellenname
  • Schriftfamilie: Schriftfamilie (Standard: Arial)
  • Schriftgröße: Schriftgröße (Standard: 12)
  • Farbe: Schriftfarbe und Iconfarbe (Standard: SYSTEM.GLOBALS.atvSymbolColor)
