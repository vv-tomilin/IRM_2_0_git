Description

The legend gives the following information about the data shown in the chart:

  • Name
  • Address
  • Aggregate
  • Type
  • Value
  • Visibility
  • Color
  • Marker 1 x-value
  • Marker 2 x-value
  • Delta x-value
  • Marker 1 y-value
  • Marker 2 y-value
  • Delta y-value
  • Settings

Some of the columns are turned off (invisible) in the standard configuration, but can be activated within the parameters.

The visibility of the data and the color of the line can be set directly in the table of the legend.

Parameters

The following common Parameters exists:

  • chart name: Unique name of the chart instance.
  • necessary right:
  • date time format: Format of the measured value on the x-axis.
  • date time delta format: Format of the time delta of the measured values on the x-axis.

Parameters for columns

In addition to the above-mentioned columns, there is also the possibility of configuring the following individual parameters:

  • name: Custom column name.
  • visibility: Visibility of the column
  • index: Determination of the order of the columns.
  • width: Column width

Parameter Appearence

  • table header fill color: Background color for the table header (hex value).
  • table footer fill color: Background color for the table footer; (Hex value).
  • highlight fill color: Color for highlighting (Hex value).
  • highlight selected color: Color for selection (Hex value).

Parameter Font

  • table header font color: Font color in the table header
  • table footer font color: Font color in the table footer.

Example

Create a display and add this display together with the "highcharts" display by drag & drop. Set the name of the chart instance as value for the parameter "chart name".
