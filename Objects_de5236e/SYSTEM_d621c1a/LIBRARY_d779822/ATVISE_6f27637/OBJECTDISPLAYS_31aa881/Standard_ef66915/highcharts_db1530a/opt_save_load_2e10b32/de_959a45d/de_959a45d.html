Beschreibung

Mit Hilfe dieses Display lässt sich die Konfiguration laden und speichern bzw. als Datei importieren und exportieren (.json-Datei).

Bevor eine Konfiguration geladen oder gespeichert werden kann muss die entsprechende Nodeadresse ausgewählt werden. Sind entsprechende Konfigurationen vorhanden, können diese nun geladen werden.

Einstellungen

Allgemein

  • Kompression: Legt fest, ob die Daten beim speichern Komprimiert werden sollen.

Konfiguration (laden/speichern)

  • Indexkonoten: Legt den Datenpunkt fest, in der die Konfiguration gespeichert wird.
  • Konfiguration speichern: Sichert die angegebene Konfiguration.
  • Laden oder Löschen: Laden oder löschen der ausgewählten Konfiguration. (Neu Laden, aktualisiert die Auswahl).
  • Konfiguration erstellen: Erstellt eine neue Konfiguration mit den angegebenen Namen.

Export/Import

  • Export: Exportiert alle Konfigurationen des Diagramms in eine json Datei.
  • Import: Importiert alle Konfigurationen fr das Diagramm aus einer json Datei.
Nachdem eine oder mehrere Konfigurationen aus einer Datei auf die entsprechende Nodeadresse importiert wurden, muss sie danach noch geladen werden um aktiv zu sein.

Parameter

Über die folgenden Parameter können die benötigten Rechte getrennt zum Laden und Speichern der Konfiguration vergeben werden:

  • Benötigtes Recht: Der Benutzer muss über dieses Recht verfügen, um die Funktion ausführen zu können.
  • Aktivierungsadresse: Stimmt der Wert dieser Adresse mit dem "Aktivierungswert" überein, ist die Funktion aktiviert.
  • Aktivierungswert: Wert, von dem die Funktion aktiviert wird.
