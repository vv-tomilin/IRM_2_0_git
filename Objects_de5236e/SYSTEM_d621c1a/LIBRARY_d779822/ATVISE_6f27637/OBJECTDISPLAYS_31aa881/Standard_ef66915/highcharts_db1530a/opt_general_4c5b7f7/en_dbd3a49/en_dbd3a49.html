Description

This display shows some general options which can be configured for the chart.

Options

  • Chart title: Title of the chart.
  • Measurement marker: Enable or disable the two measurement cursors.
  • Tooltips: Show or hide tooltips for the data points in the chart.
  • Export menu: Show or hide the export menu on the top right side.
  • Update Interval: Time between adding new data to the chart.
  • Data source: Defines the data source module which is used to import data.
  • Data mode: You can switch between mixed, live data only and archive data only.
  • Time format: Defines the time format of the data shown in the chart.
  • Downsampling: Downsampling is used to get a performance boost, especially on low-end devices. It reduces the number of shown data points regarding to the current screen and chart size.

Parameters

This control has the following appearance parameters:

  • font color: color which will be applied to the labels (default: global parameter atvFontColor)
  • fill color: left sidebar fill color (default: global parameter atvFillColor)
