Beschreibung

Über dieses Steuerelement wird das Dialogfeld "Login" geöffnet.
Die Zustände "eingeloggt"/"nicht eingeloggt" werden von unterschiedlichen Symbolen angezeigt.

Parameter

  • Darstellung
    • Symbol Login: Symbol für den Login-Status
    • Symbolfarbe Login: Füllfarbe für das Loginsymbol
    • Symbol Logout: Symbol für den Logout-Status
    • Symbolfarbe Logout: Füllfarbe für das Logoutsymbol
    • Füllfarbe: Hintergrundfarbe des Steuerelements
    • Füllfarbe wenn inaktiv: Füllfarbe der nicht aktiven Schaltfläche (Standard: dunkelgrau)
    • Konturfarbe wenn gedrückt: Farbe der Umrandung bei Betätigung der Schaltfläche
