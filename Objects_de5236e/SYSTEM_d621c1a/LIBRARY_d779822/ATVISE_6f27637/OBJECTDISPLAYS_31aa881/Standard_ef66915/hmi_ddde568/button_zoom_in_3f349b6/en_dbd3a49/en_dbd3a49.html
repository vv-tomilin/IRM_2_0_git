Description

This button increases the zoom level of the visualisation.

Parameters

  • target frame name: name of the iframe display that shall be zoomed
  • Appearance
    • symbol: the symbol of the button
    • symbol pressed: the symbol that will displayed when the button is pressed
    • fill color: background color of the control
    • stroke color pressed: color of the border when pressing the button
    • symbol color: color of the symbol
    • symbol color pressed: color of the symbol in pressed state
