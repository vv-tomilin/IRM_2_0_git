Beschreibung

Bei diesem Steuerelement handelt es sich um eine Gruppe von Schaltflächen, die alle Displays aus der Display-Hauptstruktur anzeigen, wenn der Mauszeiger auf sie gerichtet wird:

  • MAIN: Alle Displays im Ordner "DISPLAYS.MAIN"
  • USER: Alle Displays im Ordner "DISPLAYS.USER"
  • SYSTEM: Alle Displays im Ordner "DISPLAYS.SYSTEM"
  • REPORT: Alle Displays im Ordner "DISPLAYS.REPORT"

Parameter

  • Zielframe: Name des IFrames, in dem das Display geöffnet wird.
  • Darstellung
    • Konturfarbe wenn gedrückt: Farbe der Umrandung, wenn die Schaltflächen betätigt werden oder der Mauszeiger auf sie gerichtet wird
