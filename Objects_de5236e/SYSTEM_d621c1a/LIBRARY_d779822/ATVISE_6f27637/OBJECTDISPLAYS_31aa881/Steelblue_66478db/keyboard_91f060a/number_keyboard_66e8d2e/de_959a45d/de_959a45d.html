Beschreibung

Beim Zifferntastatur-Display wird auf dem Bildschirm eine Tastatur für die Eingabe von Ziffern angezeigt.

Um einfache Ziffern einzugeben, klicken Sie einmal auf die gewünschte Taste und vervollständigen die Eingabe durch Klicken auf die Taste "OK" (dabei wird die aktuelle Eingabe in den vorgegebenen Adress-Datenpunkt "Ziel" geschrieben). Mit der Taste "Löschen" können einzelne Ziffern gelöscht oder, wenn sie mindestens eine Sekunde gehalten wird, die gesamte Eingabe entfernt werden. Um abzubrechen, klicken Sie rechts in der Mitte auf die Schaltfläche "X".

Es gibt unterschiedliche Möglichkeiten, die Zifferneingabe zu begrenzen. Eine einfache Möglichkeit, eine Eingabe nur innerhalb eines gegebenen Bereichs zuzulassen, ist das Festlegen der Parameter "Minimum" und/oder "Maximum". Um die Anzahl der Ziffern vor und/oder hinter dem Dezimalkomma zu begrenzen, müssen diese Parameter definiert werden. Über den Parameter "Bestätigung der Eingabe" kann eine Bestätigungsmeldung angezeigt werden, bevor der Eingabewert geschrieben wird.

Parameter

Die nachstehenden allgemeinen Parameter sind verfügbar:

  • Einheit: Zeigt nach der Eingabe eine Einheit an.
  • Wert: anfangs angezeigter Eingabewert
  • Minimum: kleinster Eingabewert
  • Maximum: größter Eingabewert
  • Vorkommastellen: Anzahl der vor dem Dezimalkomma zugelassenen Stellen
  • Nachkommastellen: Anzahl der hinter dem Dezimalkomma zugelassenen Stellen
  • Ziel: Adresse des Zieldatenpunkts, in den die Eingabe geschrieben wird
  • Bestätigung der Eingabe: Bei der Einstellung "true" muss ein Dialogfeld mit einer Meldung bestätigt werden, um die Eingabe abzuschließen.
  • Text für die Bestätigung der Eingabe: Text, der im Dialogfeld für die Bestätigung der Eingabe verwendet wird
  • Passwortmodus: Zeigt anstatt der eingegeben Zeichen Sterne an
  • zu feuernder Trigger: Name des globalen Triggers, der gefeuert wird, wenn die Eingabe abgeschlossen wird

Beispiele

Erstellen Sie ein Display mit einer Schaltfläche, die ein Fenster mit der Tastatur öffnet, wenn auf sie geklickt wird, und fügen Sie den gewünschten Zieldatenpunkt zur Abfrage hinzu.
