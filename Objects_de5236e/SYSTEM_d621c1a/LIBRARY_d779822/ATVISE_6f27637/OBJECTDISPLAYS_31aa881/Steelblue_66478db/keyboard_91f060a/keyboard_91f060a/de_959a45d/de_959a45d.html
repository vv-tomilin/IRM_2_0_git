Beschreibung

Beim Tastatur-Display wird auf dem Bildschirm eine Tastatur für die String-Eingabe angezeigt.

Um einfachen Text einzugeben, klicken Sie einmal auf die gewünschte Taste und schließen dann die Eingabe durch Klicken auf die Taste "EN" ab (dabei wird die aktuelle Eingabe in den vorgegebenen Adress-Datenpunkt "Ziel" geschrieben). Mit der Taste "Löschen" können einzelne Zeichen gelöscht oder, wenn sie mindestens eine Sekunde gehalten wird, die gesamte Eingabe entfernt werden. Um abzubrechen, klicken Sie unten links auf die Schaltfläche "X".

Derzeit werden zwei Tastaturlayouts unterstützt (Englisch, Deutsch), die in Abhängigkeit von der Sprache des Kunden automatisch eingestellt werden.

Parameter

Die nachstehenden allgemeinen Parameter sind verfügbar:

  • Anfangswert: anfangs gesetzter Wert
  • Ziel: Adresse des Zieldatenpunkts, in den die Eingabe geschrieben wird
  • Tabstopp: für die Tabulatortaste verwendeter Tabstopp, Standard ist 1
  • Passwortmodus: Zeigt anstatt der eingegeben Zeichen Sterne an
  • zu feuernder Trigger: Name des globalen Triggers, der gefeuert wird, wenn die Eingabe abgeschlossen wird

Beispiele

Erstellen Sie ein Display mit einer Schaltfläche, die ein Fenster mit der Tastatur öffnet, wenn auf sie geklickt wird, und fügen Sie den gewünschten Zieldatenpunkt zur Abfrage hinzu.
