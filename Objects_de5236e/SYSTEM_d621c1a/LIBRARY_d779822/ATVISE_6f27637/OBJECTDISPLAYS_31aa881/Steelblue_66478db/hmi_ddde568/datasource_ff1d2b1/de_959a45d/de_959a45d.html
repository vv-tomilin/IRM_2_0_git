Beschreibung

Das Display stellt den Verbindungsstatus zwischen dem atvise Server und den Datenquellen sowie die Konfiguration der Datenquelle dar.

Im Standalone-Betrieb wird der Verbindungsstatus eines atvise Servers angezeigt, im Redundanz-Betrieb die Verbindungsstatus beider atvise Server.

Parameter

  • Datenquelle: Name der Datenquelle

Mögliche Zustände und Darstellung

  • Eine konfigurierte Verbindung wird durch eine graue Linie zwischen dem Server und dem URL-Konnektor der jeweiligen Verbindung dargestellt. Wird über eine solche auch aktuell kommuniziert, dann werden sie grün dargestellt.
  • Ist keine URL konfiguriert, wird keine Linie gezeichnet und der URL-Konnektor ist grau.
  • Kann keine Verbindung über die URL aufgebaut werden, wird keine Linie gezeichnet und der Konnektor ist rot.
  • Der Verbindungsstatus zu einer URL wird im Textfeld neben dem URL-Konnektor angegeben.
    Mögliche Werte sind "Connected", "LostConnection" und "NotConnected" wenn eine URL konfiguriert ist oder "-" wenn keine URL konfiguriert ist.
  • Der Verbindungsstatus der gesamten Datenquelle wird durch den Konnektor am Server dargestellt.
    Kann sich der Server über mindestens eine URL verbinden, ist der Konnektor grün. Ist der Konnektor rot, kann sich der Server über keine URL verbinden und es werden keine Daten empfangen.
  • Wenn im Redundanz-Betrieb zu einem atvise Server keine Verbindung aufgebaut werden kann, wird dieser mit einem grauen Rechteck überlagert.
