Description

This control is a group of buttons:

  • stop horn: button that stops the alarm sound
  • go to last alarm: opens the display of the last alarm that occured in the system
  • accept display: button that acknowledges all alarms that are located on the currently opened display
  • open alarm list: opens the alarm list display

Parameters

  • alarm list display: the display that will be opened when pressing the "open alarm list" button
  • target frame: frame in which the alarm list will be opened
  • enable sound: if set to "true", an alarm sound will be played when an alarm and the button "stop horn" are active
  • stroke color when pressed: color of the border when pressing/hovering the buttons
