Beschreibung

Bei diesem Steuerelement handelt es sich um eine Gruppe von Schaltflächen, über die Zoom-Funktionen für die Visualisierung ausgeführt werden:

  • Hineinzoomen: Schaltfläche zum Vergrößern
  • Herauszoomen: Schaltfläche zum Verkleinern
  • Zoom-Bereich: Schaltfläche, die die Bereichsvergrößerung aktiviert. Sie können den gewünschten Bereich mit der Maus auswählen, nachdem Sie die Schaltfläche aktiviert haben.
  • Zoom zurücksetzen: Stellt die Originalgröße wieder her.

Parameter

  • Zielframe: Frame, auf welchen sich das Zooming bezieht
  • Darstellung
    • Konturfarbe wenn gedrückt: Farbe der Umrandung, wenn die Schaltflächen betätigt werden oder der Mauszeiger auf sie gerichtet wird
