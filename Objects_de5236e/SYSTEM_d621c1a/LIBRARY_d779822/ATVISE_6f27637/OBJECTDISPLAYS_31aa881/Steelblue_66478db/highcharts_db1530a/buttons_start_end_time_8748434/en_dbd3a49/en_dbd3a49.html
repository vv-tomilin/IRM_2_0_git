Description

This display will be used to set the time axis.

This display can be used together with the "highcarts" display. It contains two buttons for setting begin and end time.

Parameters

The following common parameters are available:

  • chart name: Unique name of the chart instance.
  • x-axis indes: Index of the x-axis to be addressed.
  • font color: Color which will be applied to the labels (default: global parameter atvFontColor).

Examples

Create a display and add a trend display and this display by drag & drop. Set the desired chartname on both displays, they have to be identical.
