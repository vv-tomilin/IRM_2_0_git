Beschreibung

Dieses Display dient dem Umschalten zwischen den Chartkonfigurationen.

Dieses Display kann in Verbindung mit dem Display "highcharts" verwendet werden. Es enthält eine Combobox, welche dem Laden einer Konfiguration dient.

Parameter

Die nachstehenden Parameter sind verfügbar:

  • Chartname: Eindeutiger Name der Chartinstanz
  • Textfarbe: Farbe, die auf die Beschriftung angewendet wird (Standard: Globaler Parameter atvFontColor)
  • Benötigtes Recht für Export: Benötigtes Recht für Export

Beispiele

Erstellen Sie ein Display und fügen Sie ein Highcharts-Display zusammen mit diesem Display durch Ziehen und Ablegen hinzu. Definieren Sie bei beiden Displays den Chartnamen. Dieser muss identisch sein.
