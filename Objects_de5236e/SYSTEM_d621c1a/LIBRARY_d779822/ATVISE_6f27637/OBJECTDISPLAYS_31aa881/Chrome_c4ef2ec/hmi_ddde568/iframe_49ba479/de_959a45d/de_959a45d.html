Beschreibung

Der IFrame-Container ist ein Steuerelement, das auf zweierlei Art eingesetzt werden kann:

  • Einfach: Definieren eines Displays oder einer beliebigen Web-URL, das bzw. die angezeigt werden soll.
  • Erweitert: Für einen IFrame, der als Navigations-Frame verwendet wird (die Schaltflächen "Zoom" und "Display öffnen" können für Interaktionen mit dem IFrame verwendet werden)
Der erweiterte Modus wird über den Displayparameter "Funktionalität bezüglich Navigation hinzufügen" aktiviert.

Parameter

  • URL: Anzuzeigende Web-URL (oder interne URL für Ressourcen auf dem Server)
  • Zu öffnendes Display: Display aus der Visualisierung, die geladen wird
  • Name: Name des Frames (für Interaktionen mit den Schaltflächen "Zoom" und "Display öffnen", ebenso einsetzbar im Simple Dynamics Dialog unter der Aktion Fenster als Parameter "Ziel" oder auch mit der webMI.display.openDisplay Funktion). Der Name muss im Projekt eindeutig sein.
  • Funktionalität bezüglich Navigation hinzufügen (Zoom, ...): Der IFrame wird mit Triggern für die Navigation und Zoomfunktionen geladen ("erweiterter Modus").
  • Standard Frame: Auf true setzen, wenn es sich hierbei um den Standard Frame handelt.
