Beschreibung

Dieses Steuerelement zeigt die aktuelle Client-Zeit an (Stunden, Minuten, Sekunden und Millisekunden).

Parameter

  • base: Wenn keine base festgelegt wird, dann wird die aktuelle Client-Zeit angezeigt.
  • Aktualisierungsintervall: Wenn keine base festgelegt wird, dann besteht die Möglichkeit ein Aktualisierungsintervall(in ms) für die Client-Zeit zu bestimmen.
