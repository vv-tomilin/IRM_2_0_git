Beschreibung

Bei diesem Steuerelement handelt es sich um eine Gruppe von Schaltflächen:

  • Signal aus: Schaltfläche, die das Alarmsignal ausschaltet
  • Gehe zu letztem Alarm: Zeigt den letzten im System aufgetretenen Alarm an
  • Display quittieren: Schaltfläche, die alle im derzeit geöffneten Display angezeigten Alarme quittiert
  • Alarmliste öffnen: Öffnet das Dialogfeld "Alarmliste"

Parameter

  • Alarmliste: Display, das bei Betätigung der Schaltfläche "Alarmliste öffnen" geöffnet wird
  • Zielframe: Frame, in dem die Alarmliste geöffnet wird
  • Sound aktivieren: Bei "true" wird ein Alarm-Sound abgespielt, wenn ein Alarm ansteht und die Schaltfläche "Signal aus" aktiv ist
  • Konturfarbe wenn gedrückt: Farbe der Umrandung, wenn die Schaltflächen betätigt werden oder der Mauszeiger auf sie gerichtet wird
