Description

This control allows you to select a node.

In addition the trigger "valuechanged" will be fired.

An additional border indicates the current value's negative state or an additionally defined alarm. The colors and blinking intervals are defined in global lists.

Parameters

This control has the following generic parameters:

  • start address: defines the entry point of the address space
  • base: this address will be set
  • alarm address: optional additional address of an alarm to be shown.
  • enable status monitoring: switch monitoring of current value status on or off.
  • fire status trigger: instead of the status display a here defined trigger can be fired globally.
Font:
  • family: font family which will be applied to value (default: Arial)
  • size: font size which will be applied to the value (default: 12)
  • color: color which will be applied to the value (default: black)
Appearance:
  • fill color: fill color (default: grey)
  • fill color when inactive: fill color if not active (default: dark grey)
  • button fill color: button's fill color (default: black)
  • symbol fill color: symbol's fill color (default: grey)
  • stroke color pressed: the color of the stroke which will be set when the button is pressed (default: white)
  • color for context font: color for the text in the context menu (default: black)
  • fill color for context menu: fill color for the context menu (default: white)
  • stroke color for context menu: color for the stroke of the context menu (default: black)
  • stroke width for context menu: width of the border of the context menu (default: 2)
  • hover color for context menu: color for the selected item in the context menu (default: light grey)
  • focus stroke color: the color of the stroke which will be set when the button is focused (default: white)
  • status indication: specify the status' display through a defined global list (default: global parameter atvStatusIndication)
  • alarm indication: specify the alarm's display through a defined global list (default: global parameter atvAlarmIndication)
Security:
  • necessary right: the user must have this right to activate this control, i.e. to enter values
  • activation address: if the value of this address equals to "activation value", then the control will be active, i.e. can be used as input field
  • activation value: value which activates the control
NOTE: The activation of the control can be configured only by the necessary right, or only by the activation address and value or by both. In case both settings are made, the user must have the necessary right and the value of the "activation address" must equal the "activation value"; otherwise the control will not be activated.

Options
  • on demand: if on demand is set, sub level entries will be loaded when required, otherwise all entries will be loaded at once
  • maximum displayed items (0 = all): number of maximum items to be displayed per scrolling page (0 = scrolling off, default: 0)
  • show base variables: whether or not base variables shall be available to pick
  • show objects: whether or not objects shall be available to pick
  • show displays: whether or not displays shall be available to pick
  • show archives: whether or not archives shall be available to pick
  • show alarms: whether or not alarms shall be available to pick
  • show variable types: whether or not variable types shall be available to pick
  • consistency group: name of the consistency group
  • tab index: defines the order for changing between the object displays by using the tab key
  • tooltip: displays a tooltip box with the entered text (standard HTML-Elements can be entered)
