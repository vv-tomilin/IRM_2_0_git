Beschreibung

Über diese Schaltfläche wird der Wert einer Adresse erhöht oder verringert.

Darüber hinaus wird der Trigger "valuechanged" mit dem neuen Wert (= erhöhter oder verringerter Wert) aktiviert.

Parameter

Dieses Steuerelement hat die folgenden allgemeinen Parameter:

  • base: Der Wert dieser Adresse wird entweder erhöht oder verringert, je nachdem, ob "Schritt" positiv oder negativ ist.
  • Schrittweite (kann negativ sein): Der Wert, um den der Wert von "base" geändert wird. Ist er positiv, wird der Wert von "base" erhöht, andernfalls wird er verringert (Standard: 1).
  • Intervall [ms] (0 = aus): Legt fest, wie lange das Steuerelement betätigt werden muss, damit der Wert von "base" automatisch geändert wird (Standard: 150 ms, d. h., wird die Schaltfläche kontinuierlich betätigt, wird der Wert alle 150 ms geändert).
Schrift:
  • Schriftart: Schriftart, die auf die Beschriftung angewendet wird (Standard: Arial)
  • Größe: Schriftgröße, die auf die Beschriftung angewendet wird (Standard: 12)
  • Farbe:Farbe, die auf die Beschriftung angewendet wird (Standard: grau)
Darstellung:
  • Sichtbarkeit Beschriftung: Bei Einstellung auf "visible" wird der Text "+/-" als Beschriftung angezeigt; bei Einstellung auf "visible" sollte "Sichtbarkeit Symbol" auf  "hidden" eingestellt sein (Standard: visible).
  • Sichtbarkeit Symbol: Bei Einstellung auf "visible" wird das über "Symbol für Erhöhen" oder "Symbol für Verringern" definierte Symbol angezeigt; bei Einstellung auf "visible" sollte "Sichtbarkeit Beschriftung" auf "hidden" eingestellt sein (Standard: hidden).
  • Symbol für erhöhen: zu verwendendes Symbol, wenn "Sichtbarkeit Symbol" auf "visible" eingestellt ist; weitere Symbole können ausgewählt werden aus "Object Displays (ATVISE)/Symbols/20x20 (Standard: arrow_up)
  • Symbol für verringern: zu verwendendes Symbol, wenn "Sichtbarkeit Symbol" auf "visible" eingestellt und "Schritt" negativ ist; weitere Symbole können ausgewählt werden aus "Object Displays (ATVISE)/Symbols/20x20 (Standard: arrow_down)
  • Symbolfarbe: Farbe des zu verwendenden Symbols (Standard: grau)
  • Füllfarbe: Füllfarbe der Schaltfläche (Standard: schwarz)
  • Füllfarbe wenn inaktiv: Füllfarbe der nicht aktiven Schaltfläche (Standard: dunkelgrau)
  • Konturfarbe wenn gedrückt: Farbe der Kontur, die bei Betätigung der Schaltfläche eingestellt wird (Standard: weiß)
  • Fokus Konturfarbe: Farbe der Kontur, die beim Fokus der Schaltfläche eingestellt wird (Standard: weiß)
Optionen:
  • Minimum: Legt das untere Limit für den zu ändernden Wert fest, d. h., wenn der aktuelle Wert mit diesem Minimum übereinstimmt, wird er nicht weiter verringert (Standard: kein Minimum festgelegt).
  • Maximum: Legt das obere Limit für den zu ändernden Wert fest, d. h., wenn der aktuelle Wert mit diesem Maximum übereinstimmt, wird er nicht weiter erhöht (Standard: kein Maximum festgelegt).
  • Tab-Index: Bestimmt die Reihenfolge, in der mit der Tabulatortaste zwischen den einzelnen Object Displays gewechselt werden kann
  • Tooltip: Zeigt eine Tooltip Box mit dem eingegebenen Text an (Standard HTML-Elemente können auch eingegeben werden)
Sicherheit:
  • Benötigtes Recht: Der Benutzer muss über dieses Recht verfügen, um diese Schaltfläche aktivieren, d. h. betätigen zu können.
  • Aktivierungsadresse: Stimmt der Wert dieser Adresse mit dem "Aktivierungswert" überein, ist die Schaltfläche aktiv, so dass sie betätigt werden kann.
  • Aktivierungswert: Wert, von dem die Schaltfläche aktiviert wird
HINWEIS: Die Aktivierung einer Schaltfläche kann nur über das benötigte Recht konfiguriert werden oder nur über die Aktivierungsadresse und den Aktivierungswert oder über beide. Werden beide Einstellungen vorgenommen, muss der Benutzer das benötigte Recht besitzen und der Wert der "Aktivierungsadresse" muss mit dem "Aktivierungswert" identisch sein, andernfalls wird die Schaltfläche nicht aktiviert.
