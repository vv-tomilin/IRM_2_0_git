Description

This is the main display of the configuration dialog. The configuration dialog can be opened via the toolbar. It is mainly a navigation through the different options of the chosen chart.

Parameters

The following common Parameters exists:

  • chart name: This is a unique name to identify the chart instance.

Parameter representation

  • fill color: left sidebar fill color (default: global parameter atvFillColor)
  • font color: Font color (default: global parameter atvFontColor)
  • show "Preview" button (popup mode only): Activates the "Preview" button if the configuration dialog is opened in a popup.
  • show "Back" button: Activates the "Back" button if the configuration dialog is opened in a frame.

Parameter options

  • "Back" button URL: Defines the display to open on pressing the "back" button.
  • frame for "Back" button: Defines the framename to open that display.

Parameter security

In addition, the required rights for loading and saving the configuration can be assigned via the corresponding parameters:

  • necessary right: the user must have this right for the function to work
  • activation address: if the value of this address equals to "activation value", then the function will be active
  • activation value: value which activates the function
