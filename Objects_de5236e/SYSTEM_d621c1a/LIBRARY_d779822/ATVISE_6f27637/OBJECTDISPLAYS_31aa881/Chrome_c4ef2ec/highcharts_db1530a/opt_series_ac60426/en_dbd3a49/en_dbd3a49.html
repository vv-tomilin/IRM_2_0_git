Description

This display provides the management of the data sets for the chart.

Optionen

Data set to configure

  • Add: Adds a new record at the end of the record selection.
  • Data set selection: Selection area for the current record.
  • Delete selected: Deletes the selected record.

Source address (of the selected data set)

  • Type: Defines the display type (line, bar, area, etc.).
  • Visibility: Enables or disables the display of the record.
  • Address selection area: In the selection area for the address, the data point can be selected, entered or searched. (*)
  • Selectet Aggreagate: For aggregates, the information type, interval and unit are displayed.

(*) In the case of an area, two addresses must be selected - one for each address. the lower and upper line or border.

Data set configuration

  • Label: Name of the record to be displayed (default: node name).
  • Tooltip settings: Here you can define the prefix, suffix and the decimal places to be displayed.
  • z-Index: Defines the order of the data sets in the diagram.

Type configuration

The record configuration differs due to the type selection. Depending on the type, the associated X-axis and Y-axis, interpolation, color, line width, etc. can be defined. More is described on the respective subdisplay of the type.

Type configuration

Regarding to the line type the related x-axis and y-axis, the interpolation type, colors, line widths and so on can be defined. More information can be found in the particular subdisplay of the line type.
