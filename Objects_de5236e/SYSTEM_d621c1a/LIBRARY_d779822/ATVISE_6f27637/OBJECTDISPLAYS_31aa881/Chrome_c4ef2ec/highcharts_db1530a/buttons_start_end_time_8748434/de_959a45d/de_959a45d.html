Beschreibung

Dieses Display dient zur Einstellung der Zeitachse.

Dieses Display kann in Verbindung mit dem Display "highcharts" verwendet werden. Es enthält zwei Buttons zur Angabe von Anfangs- und Endzeit.

Parameter

Die nachstehenden allgemeinen Parameter sind verfügbar:

  • Diagrammname: Eindeutiger Name der Chartinstanz
  • X-Achse Index: Index der angesprochenen X-Achse
  • Textfarbe: Farbe, die auf die Beschriftung angewendet wird (Standard: Globaler Parameter atvFontColor)

Beispiele

Erstellen Sie ein Display und fügen Sie ein Highcharts-Display zusammen mit diesem Display durch Ziehen und Ablegen hinzu. Definieren Sie bei beiden Displays den Chartnamen. Dieser muss identisch sein.
